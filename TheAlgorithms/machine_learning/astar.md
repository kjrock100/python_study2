# A* 알고리즘 (A* Algorithm)

이 문서는 `astar.py` 파일에 구현된 **A* (A-Star) 탐색 알고리즘**에 대해 설명합니다.

## 개요

A* 알고리즘은 시작점에서 목표점까지의 최적 경로를 찾는 데 사용되는 그래프 탐색 알고리즘입니다. 각 노드 $n$에 대해 평가 함수 $f(n) = g(n) + h(n)$을 사용하여 가장 유망한 노드를 먼저 탐색하는 **최선 우선 탐색(Best-First Search)** 방식입니다.

- **$g(n)$**: 시작 노드에서 현재 노드 $n$까지의 실제 이동 비용.
- **$h(n)$**: 현재 노드 $n$에서 목표 노드까지의 추정 비용 (휴리스틱).

이 코드는 2차원 그리드 환경에서 A* 알고리즘을 사용하여 경로를 찾는 과정을 보여줍니다.

## 주요 클래스

### `Cell`

- **목적**: 그리드 상의 각 칸(셀)을 나타내는 클래스입니다.
- **주요 속성**:
  - `position`: 셀의 (x, y) 좌표.
  - `parent`: 경로를 역추적하기 위한 부모 셀.
  - `g`: 시작점으로부터의 실제 거리.
  - `h`: 목표점까지의 추정 거리 (휴리스틱). 이 구현에서는 유클리드 거리의 제곱을 사용합니다.
  - `f`: 총 예상 비용 (`g + h`).

### `Gridworld`

- **목적**: 탐색이 이루어지는 2차원 그리드 세계를 표현합니다.
- **주요 속성**:
  - `w`: 그리드를 나타내는 NumPy 배열.
- **주요 메서드**:
  - `get_neigbours(cell)`: 주어진 셀의 8방향(대각선 포함) 이웃 셀들을 반환합니다.

## 주요 함수: `astar`

### `astar(world, start, goal)`

- **목적**: A* 알고리즘을 실행하여 시작 셀에서 목표 셀까지의 최단 경로를 찾습니다.
- **매개변수**:
  - `world`: `Gridworld` 객체.
  - `start`: 시작 `Cell` 객체.
  - `goal`: 목표 `Cell` 객체.
- **반환값**: 시작점에서 목표점까지의 경로 좌표 리스트.
- **알고리즘 동작**:
  1. `_open` (탐색할 노드 리스트)과 `_closed` (탐색 완료된 노드 리스트)를 초기화합니다.
  2. `_open` 리스트에 시작 노드를 추가합니다.
  3. `_open` 리스트가 빌 때까지 반복합니다.
     - `_open` 리스트에서 `f` 비용이 가장 작은 노드를 현재 노드로 선택하고 `_closed`로 옮깁니다.
     - 현재 노드가 목표점이면 탐색을 종료하고, 부모 노드를 따라 경로를 역추적하여 반환합니다.
     - 현재 노드의 이웃들을 탐색하며, 각 이웃에 대해 `g`, `h`, `f` 비용을 계산하고 `_open` 리스트에 추가하거나 더 나은 경로일 경우 갱신합니다.

## 사용법

`if __name__ == "__main__":` 블록에서 5x5 그리드를 생성하고, 시작점 (0, 0)에서 목표점 (4, 4)까지의 경로를 찾은 뒤, 경로를 그리드에 `1`로 표시하여 출력합니다.

```python
world = Gridworld()
start = Cell()
start.position = (0, 0)
goal = Cell()
goal.position = (4, 4)

path = astar(world, start, goal)
print(path)
# 출력: [(0, 0), (1, 1), (2, 2), (3, 3), (4, 4)]
```
