# `decimal_to_binary.py` 코드 설명

이 문서는 `decimal_to_binary.py` 파이썬 스크립트에 포함된 `decimal_to_binary` 함수를 설명합니다. 이 스크립트는 10진수 정수를 2진수 문자열로 변환하는 기능을 구현합니다.

## 목차
1.  10진수에서 2진수로의 변환 원리
2.  함수 설명
    -   `decimal_to_binary(num)`
3.  실행 방법
4.  코드 개선 제안

## 10진수에서 2진수로의 변환 원리

10진수를 2진수로 변환하는 일반적인 방법은 **나눗셈과 나머지 연산**을 반복하는 것입니다.

1.  10진수를 2로 나눕니다.
2.  이때 발생하는 **나머지**(0 또는 1)가 2진수의 가장 낮은 자리 숫자가 됩니다.
3.  **몫**을 다시 2로 나누고, 나머지를 다음 자리 숫자로 사용합니다.
4.  몫이 0이 될 때까지 이 과정을 반복합니다.
5.  얻어진 나머지들을 역순으로 이어 붙이면 최종 변환 결과가 됩니다.

**예시 (7을 2진수로 변환):**
-   7 / 2 = 몫 3, 나머지 1
-   3 / 2 = 몫 1, 나머지 1
-   1 / 2 = 몫 0, 나머지 1
-   나머지를 역순으로 결합: `111`

## 함수 설명

### `decimal_to_binary(num: int) -> str`

10진수 정수를 입력받아 해당하는 2진수 문자열을 반환합니다.

-   **알고리즘**:
    1.  **유효성 검사**: 입력값 `num`이 `float`이나 `str` 타입이면 `TypeError`를 발생시킵니다.
    2.  **0 처리**: `num`이 0이면, "0b0"을 즉시 반환합니다.
    3.  **음수 처리**: `num`이 음수이면, `negative` 플래그를 `True`로 설정하고 `num`을 양수로 바꿉니다.
    4.  **변환**:
        -   `while` 루프를 사용하여 `num`이 0보다 클 동안 반복합니다.
        -   `num % 2`를 사용하여 나머지를 구하고, 결과 리스트(`binary`)의 맨 앞에 추가합니다.
        -   `num >>= 1` (오른쪽 비트 시프트) 연산을 통해 `num`을 2로 나눈 몫으로 업데이트합니다.
    5.  **결과 반환**: 변환된 숫자 리스트를 문자열로 합치고, "0b" 또는 "-0b" 접두사를 붙여 최종 2진수 표현 문자열을 만들어 반환합니다.

```python
>>> decimal_to_binary(35)
'0b100011'
>>> decimal_to_binary(-2)
'-0b10'
```

## 코드 개선 제안

1.  **파이썬 내장 함수 사용**: 이 변환은 파이썬의 내장 함수 `bin()`을 사용하여 훨씬 간단하게 구현할 수 있습니다. `bin(number)`는 정수를 "0b" 접두사가 붙은 2진수 문자열로 직접 변환해 줍니다. 교육적인 목적의 구현으로는 훌륭하지만, 실제 코드에서는 내장 함수를 사용하는 것이 더 간결하고 효율적입니다.

    ```python
    # 내장 함수를 사용한 개선 제안 예시
    def decimal_to_binary_fast(num: int) -> str:
        """
        Convert an integer to its binary representation using Python's built-in bin().
        """
        if not isinstance(num, int):
            raise TypeError("Input must be an integer.")
        return bin(num)
    ```

2.  **입력 유효성 검사 개선**: 현재 `isinstance`를 사용하여 `float`과 `str`을 명시적으로 거부하고 있습니다. `if not isinstance(num, int):`와 같이 정수가 아닌 모든 타입을 한 번에 거부하는 것이 더 간결할 수 있습니다. (단, `bool` 타입은 `int`의 하위 클래스이므로 `isinstance(num, bool)` 체크를 추가하여 `True`/`False`가 `1`/`0`으로 처리되는 것을 막는 것이 더 안전합니다.)

3.  **코드 중복**: 이 스크립트는 `decimal_to_binary_recursion.py`와 거의 동일한 기능을 수행합니다. 하나는 반복(iteration)으로, 다른 하나는 재귀(recursion)로 구현되어 있어 두 가지 접근 방식을 비교하는 데 유용합니다.
