## 10진수에서 8진수로의 변환 원리

10진수를 8진수로 변환하는 일반적인 방법은 **나눗셈과 나머지 연산**을 반복하는 것입니다.

1.  10진수를 8로 나눕니다.
2.  이때 발생하는 **나머지**(0~7)가 8진수의 가장 낮은 자리 숫자가 됩니다.
3.  **몫**을 다시 8로 나누고, 나머지를 다음 자리 숫자로 사용합니다.
4.  몫이 0이 될 때까지 이 과정을 반복합니다.
5.  얻어진 나머지들을 역순으로 이어 붙이면 최종 변환 결과가 됩니다.

**예시 (65를 8진수로 변환):**
-   65 / 8 = 몫 8, 나머지 1
-   8 / 8 = 몫 1, 나머지 0
-   1 / 8 = 몫 0, 나머지 1
-   나머지를 역순으로 결합: `101` -> `0o101`

## 함수 설명

### `decimal_to_octal(num: int) -> str`

10진수 정수를 입력받아 해당하는 8진수 문자열을 반환합니다.

-   **알고리즘**:
    1.  `while` 루프를 사용하여 `num`이 0보다 클 동안 반복합니다.
    2.  `num % 8`을 사용하여 나머지를 구합니다.
    3.  이 나머지를 10의 거듭제곱(`10^counter`)과 곱하여 `octal` 변수에 더합니다. 이 과정은 8진수 각 자리의 숫자를 10진수 시스템에서 올바른 위치에 배치하는 것과 같습니다. (예: 8진수 101은 10진수 101로 계산됨)
    4.  `num`을 8로 나눈 몫으로 업데이트합니다.
    5.  최종적으로 계산된 `octal` 값을 "0o" 접두사가 붙은 문자열로 변환하여 반환합니다.

> **주의**: 이 알고리즘은 8진수 값을 10진수 시스템 내에서 계산하는 독특한 방식을 사용합니다. 예를 들어, 8진수 '17'을 10진수 숫자 `17`로 계산합니다. 이는 표준적인 문자열 기반 변환 방식과 다릅니다.

## 코드 개선 제안

1.  **파이썬 내장 함수 사용**: 이 변환은 파이썬의 내장 함수 `oct()`를 사용하여 훨씬 간단하고 정확하게 구현할 수 있습니다. `oct(number)`는 정수를 "0o" 접두사가 붙은 8진수 문자열로 직접 변환해 줍니다. 교육적인 목적의 구현으로는 흥미롭지만, 실제 코드에서는 내장 함수를 사용하는 것이 더 간결하고 효율적입니다.

    ```python
    # 내장 함수를 사용한 개선 제안 예시
    def decimal_to_octal_fast(num: int) -> str:
        """
        Convert an integer to its octal representation using Python's built-in oct().
        """
        if not isinstance(num, int):
            raise TypeError("Input must be an integer.")
        return oct(num)
    ```

2.  **알고리즘 단순화**: 현재 알고리즘은 8진수 결과를 10진수 숫자로 중간에 표현하는 방식이라 혼동을 줄 수 있습니다. 나머지 값을 문자열로 바로 변환하여 앞에 추가하는 방식이 더 직관적이고 일반적입니다.

    ```python
    # 문자열 기반 알고리즘 개선 제안 예시
    def decimal_to_octal_str(num: int) -> str:
        if num == 0:
            return "0o0"
        
        octal_str = ""
        while num > 0:
            num, remainder = divmod(num, 8)
            octal_str = str(remainder) + octal_str
            
        return "0o" + octal_str
    ```

3.  **음수 및 유효성 검사**: 현재 코드는 양의 정수만 처리합니다. `decimal_to_binary.py`와 같이 음수 입력을 처리하고, `str`이나 `float` 같은 유효하지 않은 타입에 대한 예외 처리를 추가하면 함수의 안정성이 향상됩니다.
