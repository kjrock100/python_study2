# 너비 우선 탐색 최단 경로 (BFS Shortest Path)

이 문서는 `bfs_shortest_path.py` 파일에 구현된 **너비 우선 탐색(BFS)**을 이용한 최단 경로 찾기 알고리즘에 대해 설명합니다.

## 개요

가중치가 없는 그래프에서 두 노드 사이의 최단 경로를 찾는 가장 일반적인 방법은 너비 우선 탐색(BFS)을 사용하는 것입니다. 이 코드는 경로 자체를 반환하는 함수와 경로의 길이(간선 수)를 반환하는 함수 두 가지를 제공합니다.

## 주요 함수

### `bfs_shortest_path(graph, start, goal)`

- **목적**: 시작 노드(`start`)에서 목표 노드(`goal`)까지의 최단 경로를 노드들의 리스트로 반환합니다.
- **매개변수**:
  - `graph`: 인접 리스트 형태의 딕셔너리.
  - `start`: 시작 노드.
  - `goal`: 목표 노드.
- **반환값**: 최단 경로를 구성하는 노드 리스트 (예: `['A', 'B', 'D']`). 경로가 없으면 빈 리스트 `[]`를 반환합니다.
- **동작 원리**:
  - 큐(Queue)에 경로(Path) 자체를 저장합니다. `[[start]]`로 시작.
  - 큐에서 경로를 하나씩 꺼내 마지막 노드의 이웃을 확인합니다.
  - 이웃 노드를 포함한 새로운 경로를 만들어 큐에 추가합니다.
  - 목표 노드에 도달하면 즉시 해당 경로를 반환합니다. (BFS 특성상 가장 먼저 도달한 경로가 최단 경로임이 보장됨)

### `bfs_shortest_path_distance(graph, start, target)`

- **목적**: 시작 노드(`start`)에서 목표 노드(`target`)까지의 최단 거리(간선 개수)를 반환합니다.
- **매개변수**:
  - `graph`: 인접 리스트 형태의 딕셔너리.
  - `start`: 시작 노드.
  - `target`: 목표 노드.
- **반환값**: 최단 거리 (정수). 경로가 없거나 노드가 그래프에 없으면 `-1`을 반환합니다.
- **동작 원리**:
  - `dist` 딕셔너리를 사용하여 시작점으로부터 각 노드까지의 거리를 기록합니다.
  - 큐에는 노드만 저장하며, 방문하지 않은 이웃 노드로 이동할 때마다 거리를 1씩 증가시킵니다.

## 사용법

`if __name__ == "__main__":` 블록에서 예제 그래프(`demo_graph`)를 사용하여 함수를 호출하는 방법을 확인할 수 있습니다.

```python
demo_graph = {
    "A": ["B", "C", "E"],
    "B": ["A", "D", "E"],
    ...
}

print(bfs_shortest_path(demo_graph, "G", "D"))
# 출력: ['G', 'C', 'A', 'B', 'D']

print(bfs_shortest_path_distance(demo_graph, "G", "D"))
# 출력: 4
```
