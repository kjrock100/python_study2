# 너비 우선 탐색 2 (Breadth First Search 2)

이 문서는 `breadth_first_search_2.py` 파일에 구현된 **너비 우선 탐색(BFS)** 알고리즘에 대해 설명합니다.

## 개요

너비 우선 탐색(BFS)은 그래프 탐색 알고리즘 중 하나로, 시작 정점에서 가까운 정점부터 먼저 방문하고 멀리 떨어져 있는 정점을 나중에 방문하는 순회 방법입니다. 큐(Queue) 자료구조를 사용하여 구현됩니다.

## 주요 데이터 및 함수

### `G` (그래프 데이터)
- 딕셔너리 형태로 정의된 예제 그래프입니다.
- 키(Key)는 정점(Vertex)을, 값(Value)은 해당 정점과 연결된 이웃 정점들의 리스트를 나타냅니다.

### `breadth_first_search(graph: dict, start: str) -> set[str]`

- **목적**: 주어진 시작 정점(`start`)에서부터 그래프(`graph`)를 너비 우선 탐색하여 방문한 모든 정점들의 집합을 반환합니다.
- **매개변수**:
  - `graph`: 인접 리스트 형태의 그래프 딕셔너리.
  - `start`: 탐색을 시작할 정점의 이름(문자열).
- **반환값**: 방문한 정점들이 포함된 집합(`set`).
- **알고리즘 동작 원리**:
  1. `explored` 집합을 생성하고 시작 정점을 추가합니다.
  2. `queue`를 생성하고 시작 정점을 넣습니다.
  3. 큐가 빌 때까지 다음을 반복합니다:
     - 큐에서 정점 `v`를 하나 꺼냅니다.
     - `v`의 모든 이웃 `w`에 대해:
       - 만약 `w`가 아직 방문하지 않은 정점(`explored`에 없음)이라면:
         - `explored`에 추가합니다.
         - `queue`에 `w`를 넣습니다.

## 사용법

`if __name__ == "__main__":` 블록에서 `doctest`를 실행하여 코드를 검증하고, 예제 그래프 `G`에 대해 'A'에서 시작하는 BFS 결과를 출력합니다.

```python
# 실행 결과 예시
# 순서는 집합(set)이므로 보장되지 않지만 모든 노드가 포함됨
{'A', 'B', 'C', 'D', 'E', 'F'}
```
