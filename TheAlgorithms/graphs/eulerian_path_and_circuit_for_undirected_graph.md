# 무향 그래프의 오일러 경로 및 회로 (Eulerian Path and Circuit for Undirected Graph)

이 문서는 `eulerian_path_and_circuit_for_undirected_graph.py` 파일에 구현된 **무향 그래프에서의 오일러 경로 및 오일러 회로** 탐색 알고리즘에 대해 설명합니다.

## 개요

- **오일러 경로 (Eulerian Path)**: 그래프의 모든 간선을 정확히 한 번씩 방문하는 경로입니다.
- **오일러 회로 (Eulerian Circuit)**: 오일러 경로이면서 시작 정점과 끝 정점이 같은 경로입니다.

이 코드는 각 정점의 차수(Degree)를 계산하여 오일러 경로/회로 존재 여부를 판단하고, DFS(깊이 우선 탐색)를 사용하여 해당 경로를 찾습니다.

## 주요 함수

### `check_circuit_or_path(graph, max_node)`

- **목적**: 그래프가 오일러 경로를 가지는지, 오일러 회로를 가지는지, 아니면 둘 다 아닌지를 판별합니다.
- **원리**:
  - 그래프의 모든 정점에 대해 차수(연결된 간선의 수)를 셉니다.
  - **홀수 차수 정점이 0개**: 오일러 회로가 존재합니다. (반환값: 1)
  - **홀수 차수 정점이 2개**: 오일러 경로가 존재합니다. (반환값: 2)
  - **그 외**: 오일러 경로/회로가 존재하지 않습니다. (반환값: 3)
- **반환값**: `(상태 코드, 홀수 차수 정점 인덱스)` 튜플.

### `dfs(u, graph, visited_edge, path=None)`

- **목적**: 오일러 경로를 찾기 위해 그래프를 탐색합니다.
- **매개변수**:
  - `u`: 현재 정점.
  - `graph`: 인접 리스트 형태의 그래프.
  - `visited_edge`: 간선 방문 여부를 저장하는 2차원 배열 (인접 행렬 형태).
  - `path`: 현재까지의 경로 리스트.
- **동작**:
  - 현재 정점 `u`를 경로에 추가합니다.
  - `u`의 이웃 정점 `v`를 순회하며, 아직 방문하지 않은 간선 `(u, v)`라면 방문 처리하고 재귀적으로 `dfs`를 호출합니다.
  - **참고**: 이 구현은 단순 DFS를 사용하므로, 간선 선택 순서에 따라 복잡한 그래프에서는 유효한 오일러 경로를 찾지 못할 수도 있습니다 (Fleury 알고리즘이나 Hierholzer 알고리즘의 완전한 구현은 아님).

### `check_euler(graph, max_node)`

- **목적**: 전체 로직을 조율하는 메인 함수입니다.
- **동작**:
  1. `check_circuit_or_path`를 호출하여 그래프의 상태를 확인하고 출력합니다.
  2. 오일러 경로/회로가 존재하지 않으면 종료합니다.
  3. 시작 정점을 결정합니다.
     - 오일러 경로인 경우: 홀수 차수를 가진 정점(`odd_node`)에서 시작합니다.
     - 오일러 회로인 경우: 임의의 정점(여기서는 1)에서 시작합니다.
  4. `dfs`를 호출하여 경로를 탐색하고 결과를 출력합니다.

## 사용법

`main()` 함수에 정의된 예제 그래프(`G1` ~ `G5`)를 통해 동작을 확인할 수 있습니다.

```python
G1 = {1: [2, 3, 4], 2: [1, 3], 3: [1, 2], 4: [1, 5], 5: [4]}
max_node = 10
check_euler(G1, max_node)
```
