# 다익스트라 알고리즘 (Dijkstra's Algorithm)

이 문서는 `dijkstra_algorithm.py` 파일에 구현된 **다익스트라 최단 경로 알고리즘**에 대해 설명합니다. 이 구현은 외부 라이브러리 없이 최소 힙(Min Heap) 기반의 우선순위 큐를 직접 구현하여 사용합니다.

## 개요

다익스트라 알고리즘은 가중치가 있는 그래프(음수 가중치 없음)에서 특정 시작 정점으로부터 다른 모든 정점까지의 최단 경로를 찾는 알고리즘입니다.

## 주요 클래스

### `PriorityQueue`

- **목적**: 최소 힙(Min Heap)을 기반으로 한 우선순위 큐를 구현합니다. 다익스트라 알고리즘에서 현재 가장 거리가 짧은 노드를 효율적으로 추출하기 위해 사용됩니다.
- **주요 메서드**:
  - `insert(tup)`: 노드와 거리 정보를 큐에 삽입합니다.
  - `extract_min()`: 우선순위 큐에서 최소값(가장 짧은 거리의 노드)을 제거하고 반환합니다.
  - `decrease_key(tup, new_d)`: 특정 노드의 거리가 갱신되었을 때, 힙 내에서 위치를 조정합니다.
  - `min_heapify(idx)`: 힙 속성을 유지하기 위해 트리를 재구성합니다.

### `Graph`

- **목적**: 그래프를 표현하고 다익스트라 알고리즘을 실행합니다.
- **속성**:
  - `adjList`: 인접 리스트 형태로 그래프를 저장합니다. `u -> [(v, w), ...]`
  - `dist`: 시작 정점으로부터의 최단 거리를 저장하는 리스트.
  - `par`: 경로 역추적을 위해 각 노드의 부모 노드를 저장하는 리스트.
- **주요 메서드**:
  - `add_edge(u, v, w)`: 두 정점 `u`, `v` 사이에 가중치 `w`인 간선을 추가합니다. (무향 그래프로 처리됨)
  - `dijkstra(src)`: 시작 정점 `src`로부터 다른 모든 정점까지의 최단 거리를 계산합니다.
    - 우선순위 큐를 사용하여 방문하지 않은 노드 중 거리가 가장 짧은 노드를 선택하고, 인접한 노드들의 거리를 갱신(Relaxation)합니다.
  - `show_path(src, dest)`: `dijkstra` 실행 후, `src`에서 `dest`까지의 경로와 총 비용을 출력합니다. `par` 리스트를 역추적하여 경로를 구합니다.

## 사용법

`if __name__ == "__main__":` 블록에서 그래프를 생성하고 간선을 추가한 뒤 알고리즘을 실행하는 예시를 확인할 수 있습니다.

```python
graph = Graph(9)
graph.add_edge(0, 1, 4)
# ... 간선 추가 ...
graph.show_graph()
graph.dijkstra(0)
graph.show_path(0, 4)
```
