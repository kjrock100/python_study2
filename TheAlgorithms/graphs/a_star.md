# A* 알고리즘 (A* Algorithm)

이 문서는 `a_star.py` 파일에 구현된 **A* (A-Star) 탐색 알고리즘**에 대해 설명합니다. 이 알고리즘은 주어진 2차원 그리드 상에서 시작점부터 목표 지점까지의 최단 경로를 찾는 데 사용됩니다.

## 개요

A* 알고리즘은 다익스트라 알고리즘(Dijkstra's Algorithm)과 탐욕적 최선 우선 탐색(Greedy Best-First Search)의 장점을 결합한 알고리즘입니다. 각 노드 $n$에 대해 평가 함수 $f(n) = g(n) + h(n)$을 사용하여 탐색 순서를 결정합니다.
- $g(n)$: 시작 노드에서 현재 노드 $n$까지의 실제 이동 비용.
- $h(n)$: 현재 노드 $n$에서 목표 노드까지의 추정 비용 (휴리스틱).

## 주요 변수

- `DIRECTIONS`: 이동 가능한 4가지 방향을 정의한 리스트입니다. `[행 변화량, 열 변화량]` 형태로 저장되어 있습니다.
  - `[-1, 0]`: 위쪽 행으로 이동
  - `[0, -1]`: 왼쪽 열로 이동
  - `[1, 0]`: 아래쪽 행으로 이동
  - `[0, 1]`: 오른쪽 열로 이동
  - *(참고: 코드 내 주석에 적힌 방향(left, down 등)은 일반적인 행렬 인덱싱 기준과는 다소 차이가 있을 수 있습니다.)*

## 주요 함수: `search`

### `search(grid, init, goal, cost, heuristic)`

- **목적**: 그리드에서 장애물을 피해 목표까지의 최단 경로를 찾습니다.
- **매개변수**:
  - `grid`: 0(이동 가능)과 1(장애물)로 구성된 2차원 리스트.
  - `init`: 시작 좌표 `[행, 열]`.
  - `goal`: 목표 좌표 `[행, 열]`.
  - `cost`: 한 칸 이동할 때 드는 비용.
  - `heuristic`: 각 셀에서 목표까지의 추정 비용이 미리 계산된 2차원 리스트.
- **반환값**:
  - `path`: 시작점부터 목표점까지의 경로 좌표 리스트.
  - `action`: 각 셀에서 어떤 방향으로 이동했는지를 기록한 2차원 리스트.

### 알고리즘 동작 원리

1. **초기화**:
   - `closed`: 방문 여부를 체크하는 그리드. 시작점을 방문 처리합니다.
   - `action`: 경로 복원을 위해 이동 방향(인덱스)을 저장할 그리드.
   - `cell`: 탐색할 노드들을 저장하는 리스트 (우선순위 큐 역할). `[f, g, x, y]` 형태로 저장됩니다.

2. **탐색 루프**:
   - `cell` 리스트에서 $f$ 값(총 비용)이 가장 낮은 노드를 꺼냅니다.
   - 현재 노드가 목표 지점이면 탐색을 종료합니다.
   - 목표가 아니라면, 4방향으로 이동 가능한 이웃 노드들을 확인합니다.
     - 그리드 범위를 벗어나지 않고, 장애물이 아니며, 방문하지 않은 노드라면:
       - 새로운 $g$ 비용과 $f$ 비용을 계산하여 `cell`에 추가하고, `closed`에 방문 표시를 합니다.
       - `action` 그리드에 이동 방향 인덱스를 저장합니다.

3. **경로 복원**:
   - 목표 지점에서 시작하여 `action` 그리드에 저장된 방향 정보를 이용해 역추적하며 시작점까지 이동합니다.
   - 이 과정을 통해 최단 경로 좌표들을 `path` 리스트에 담아 반환합니다.

## 사용법

`if __name__ == "__main__":` 블록에서는 0과 1로 구성된 예제 그리드를 정의하고, 맨해튼 거리(Manhattan Distance)를 휴리스틱으로 사용하여 경로를 찾는 과정을 보여줍니다.
