# 이분 그래프 확인 (DFS) (Check Bipartite Graph DFS)

이 문서는 `check_bipartite_graph_dfs.py` 파일에 구현된 **DFS(깊이 우선 탐색)**를 이용한 이분 그래프 판별 알고리즘에 대해 설명합니다.

## 개요

**이분 그래프(Bipartite Graph)**란 그래프의 모든 정점을 두 개의 서로 겹치지 않는 집합(U, V)으로 나누었을 때, 그래프의 모든 간선이 U의 정점과 V의 정점을 연결하는 그래프를 의미합니다. 즉, 인접한 정점끼리는 서로 다른 색으로 칠할 수 있는 그래프(2-Colorable)입니다.

## 주요 함수: `check_bipartite_dfs`

### `check_bipartite_dfs(graph)`

- **목적**: 주어진 그래프가 이분 그래프인지 확인합니다.
- **매개변수**:
  - `graph`: 인접 리스트 형태의 딕셔너리 (예: `{0: [1, 3], 1: [0, 2], ...}`). 정점은 0부터 N-1까지의 정수여야 합니다.
- **반환값**: 이분 그래프이면 `True`, 아니면 `False`.

### 알고리즘 동작 원리

1. **초기화**:
   - `visited`: 각 정점의 방문 여부를 저장하는 리스트.
   - `color`: 각 정점의 색상을 저장하는 리스트 (0 또는 1).

2. **DFS 탐색 및 색칠 (`dfs` 내부 함수)**:
   - 현재 정점 `v`를 방문 처리하고 색상 `c`를 할당합니다.
   - 인접한 정점 `u`에 대해 아직 방문하지 않았다면, 색상을 반전(`1 - c`)시켜 재귀적으로 DFS를 호출합니다.

3. **전체 정점 순회**:
   - 그래프가 연결되어 있지 않은 경우(Disconnected Graph)를 대비하여, 모든 정점을 확인하며 방문하지 않은 정점에서 DFS를 시작합니다.

4. **유효성 검사**:
   - 모든 정점에 대해 색칠이 완료된 후, 그래프의 모든 간선을 확인합니다.
   - 만약 인접한 두 정점 `i`와 `j`가 같은 색(`color[i] == color[j]`)을 가지고 있다면, 이분 그래프 조건을 만족하지 못하므로 `False`를 반환합니다.
   - 모든 검사를 통과하면 `True`를 반환합니다.

## 사용법

코드 하단에 예제 그래프 데이터가 포함되어 있으며, 함수를 호출하여 결과를 확인할 수 있습니다.

```python
graph = {
    0: [1, 3],
    1: [0, 2],
    2: [1, 3],
    3: [0, 2],
    4: []
}
print(check_bipartite_dfs(graph))
# 출력: True
```
