# 단절점 (Articulation Points)

이 문서는 `articulation_points.py` 파일에 구현된 **단절점(Articulation Points)** 찾기 알고리즘에 대해 설명합니다.

## 개요

그래프 이론에서 단절점(또는 절단점)이란 무향 그래프에서 해당 정점과 연결된 간선들을 제거했을 때, 그래프의 연결 요소(Connected Component) 개수가 증가하는 정점을 말합니다. 즉, 이 점을 제거하면 그래프가 두 개 이상으로 나뉘게 됩니다.

## 주요 함수: `computeAP`

### `computeAP(l)`

- **목적**: 주어진 무향 그래프에서 모든 단절점을 찾아 출력합니다.
- **매개변수**:
  - `l`: 그래프의 인접 리스트 (Adjacency List). 딕셔너리 또는 리스트 형태일 수 있습니다.
- **동작 원리**:
  - **DFS (깊이 우선 탐색)**를 기반으로 합니다.
  - 각 정점에 대해 `low` 값을 계산합니다. 여기서 `low[u]`는 `u`를 루트로 하는 서브트리에서 역방향 간선(Back-edge)을 통해 도달할 수 있는 정점 중 가장 작은 인덱스(또는 우선순위)를 의미합니다.
  - **단절점 판별 조건**:
    1. **루트 노드인 경우**: DFS 트리에서 자식 노드가 2개 이상이면 단절점입니다. (`outEdgeCount > 1`)
    2. **루트 노드가 아닌 경우**: 정점 `u`의 자식 정점 `v`에 대해, `v`에서 `u`의 조상으로 가는 역방향 간선이 없다면(`low[v] >= u`), `u`는 단절점입니다.

### 코드 구조

1. **초기화**:
   - `low`: 각 정점의 low-link 값을 저장하는 리스트.
   - `visited`: 방문 여부를 저장하는 리스트.
   - `isArt`: 단절점 여부를 저장하는 리스트.

2. **DFS 함수 (`dfs`)**:
   - 재귀적으로 그래프를 탐색합니다.
   - `visited`를 갱신하고 `low` 값을 초기화합니다.
   - 인접한 정점들을 순회하며:
     - 부모 노드는 무시합니다.
     - 방문하지 않은 노드라면 재귀 호출을 수행하고, 자식의 `low` 값을 이용해 현재 노드의 `low` 값을 갱신합니다.
     - 단절점 조건을 검사하여 `isArt`를 `True`로 설정합니다.
     - 이미 방문한 노드라면(역방향 간선), 해당 노드의 인덱스를 이용해 `low` 값을 갱신합니다.

3. **결과 출력**:
   - 모든 정점에 대해 DFS를 수행한 후(연결되지 않은 그래프일 수 있으므로), `isArt`가 `True`인 정점들의 인덱스를 출력합니다.

## 사용법

코드 하단에 예제 그래프 데이터가 포함되어 있으며, 함수를 호출하여 결과를 확인할 수 있습니다.

```python
data = {
    0: [1, 2],
    1: [0, 2],
    ...
}
computeAP(data)
```
