# 칸의 알고리즘을 이용한 위상 정렬 (Topological Sort using Kahn's Algorithm)

이 문서는 `kahns_algorithm_topo.py` 파일에 구현된 **칸의 알고리즘(Kahn's Algorithm)**을 사용하여 **유향 비순환 그래프(DAG)**의 위상 정렬을 수행하는 방법에 대해 설명합니다.

## 개요

위상 정렬(Topological Sort)은 유향 그래프의 정점들을 변의 방향을 거스르지 않도록 나열하는 것입니다. 칸의 알고리즘은 진입 차수(Indegree)와 큐(Queue)를 사용하여 이를 구현합니다.

## 주요 함수: `topologicalSort`

### `topologicalSort(graph)`

- **목적**: 주어진 그래프에 대해 위상 정렬을 수행하고 결과를 출력합니다. 사이클이 존재할 경우 이를 감지합니다.
- **매개변수**:
  - `graph`: 인접 리스트 형태의 딕셔너리. (Key: 정점 인덱스, Value: 이웃 정점 리스트)
- **동작 원리**:
  1. **진입 차수(Indegree) 계산**: 모든 정점에 대해 들어오는 간선의 수를 계산합니다.
  2. **큐 초기화**: 진입 차수가 0인 정점들을 큐에 넣습니다. 이는 선행 작업이 없는 정점들을 의미합니다.
  3. **BFS 탐색**:
     - 큐에서 정점 `vertex`를 꺼내고 위상 정렬 결과 리스트 `topo`에 추가합니다.
     - 방문한 정점 수 `cnt`를 증가시킵니다.
     - `vertex`와 연결된 모든 이웃 정점 `x`에 대해:
       - `x`의 진입 차수를 1 감소시킵니다. (선행 정점 `vertex`가 처리되었으므로)
       - 만약 `x`의 진입 차수가 0이 되면 큐에 추가합니다.
  4. **사이클 감지 및 결과 출력**:
     - 모든 정점을 방문하기 전에 큐가 비어버리면(`cnt != len(graph)`), 그래프에 사이클이 존재한다는 의미입니다.
     - 사이클이 없으면 위상 정렬된 리스트 `topo`를 출력합니다.

## 사용법

코드 하단에 예제 그래프 데이터가 포함되어 있으며, 함수를 호출하여 결과를 확인할 수 있습니다.

```python
# 그래프 정의 (인접 리스트)
graph = {
    0: [1, 2],
    1: [3],
    2: [3],
    3: [4, 5],
    4: [],
    5: []
}

# 함수 실행
topologicalSort(graph)
```

### 실행 결과 예시
위 그래프의 위상 정렬 결과 중 하나는 다음과 같습니다.

```text
[0, 1, 2, 3, 4, 5]
```
