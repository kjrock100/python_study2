# 너비 우선 탐색 (Breadth First Search)

이 문서는 `breadth_first_search.py` 파일에 구현된 **너비 우선 탐색(BFS)** 알고리즘에 대해 설명합니다.

## 개요

너비 우선 탐색(BFS)은 시작 정점으로부터 가까운 정점들을 먼저 방문하고, 멀리 있는 정점들을 나중에 방문하는 그래프 탐색 방법입니다. 이 구현에서는 큐(Queue) 자료구조를 사용하여 탐색 순서를 관리합니다.

## 주요 클래스: `Graph`

### `__init__(self)`
- **목적**: 그래프 객체를 초기화합니다.
- **속성**:
  - `vertices`: 정점과 그 이웃 정점들의 리스트를 매핑하는 딕셔너리입니다. (인접 리스트 방식)

### `add_edge(self, from_vertex: int, to_vertex: int)`
- **목적**: 두 정점 사이에 간선을 추가합니다.
- **특징**: 이 구현에서는 유향 그래프(Directed Graph) 형태로 간선을 추가합니다. 즉, `from_vertex`에서 `to_vertex`로 가는 방향만 저장됩니다.

### `print_graph(self)`
- **목적**: 현재 그래프의 인접 리스트 상태를 출력합니다.
- **출력 형식**: `정점 : 이웃1 -> 이웃2 ...`

### `bfs(self, start_vertex: int) -> set[int]`
- **목적**: 주어진 `start_vertex`에서 시작하여 그래프를 너비 우선으로 탐색합니다.
- **반환값**: 방문한 모든 정점들의 집합(`set`)을 반환합니다.
- **알고리즘 동작**:
  1. `visited` 집합을 생성하고 시작 정점을 추가합니다.
  2. `queue`를 생성하고 시작 정점을 넣습니다.
  3. 큐가 빌 때까지 다음을 반복합니다:
     - 큐에서 정점을 하나 꺼냅니다.
     - 해당 정점의 이웃 정점들을 순회하며, 아직 방문하지 않은 정점이라면 `visited`에 추가하고 큐에 넣습니다.

## 사용법

`if __name__ == "__main__":` 블록에서 실행 예시를 확인할 수 있습니다.

```python
g = Graph()
g.add_edge(0, 1)
g.add_edge(0, 2)
g.add_edge(1, 2)
g.add_edge(2, 0)
g.add_edge(2, 3)
g.add_edge(3, 3)

g.print_graph()
# 출력:
# 0  :  1 -> 2
# 1  :  2
# 2  :  0 -> 3
# 3  :  3

print(sorted(g.bfs(2)))
# 출력: [0, 1, 2, 3]
```
