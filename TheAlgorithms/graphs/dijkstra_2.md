# 다익스트라 알고리즘 2 (Dijkstra's Algorithm 2)

이 문서는 `dijkstra_2.py` 파일에 구현된 **다익스트라(Dijkstra) 최단 경로 알고리즘**에 대해 설명합니다.

## 개요

다익스트라 알고리즘은 음의 가중치가 없는 그래프에서 한 정점(Source)에서 다른 모든 정점까지의 최단 경로를 구하는 알고리즘입니다. 이 구현에서는 **인접 행렬(Adjacency Matrix)**을 사용하여 그래프를 표현합니다.

## 주요 함수

### `minDist(mdist, vset, V)`

- **목적**: 최단 경로 트리에 아직 포함되지 않은 정점들 중에서, 출발점에서 가장 가까운(거리가 최소인) 정점의 인덱스를 반환합니다.
- **매개변수**:
  - `mdist`: 출발점에서 각 정점까지의 현재 최단 거리 리스트.
  - `vset`: 각 정점이 최단 경로 트리에 포함되었는지(방문했는지) 여부를 나타내는 불리언 리스트.
  - `V`: 정점의 개수.
- **반환값**: 최소 거리를 가진 정점의 인덱스 (`minInd`).

### `Dijkstra(graph, V, src)`

- **목적**: 다익스트라 알고리즘을 수행하여 최단 거리를 계산하고 출력합니다.
- **매개변수**:
  - `graph`: 인접 행렬로 표현된 그래프 (`graph[u][v]`는 u에서 v로 가는 가중치).
  - `V`: 정점의 개수.
  - `src`: 출발 정점의 인덱스.
- **알고리즘 동작**:
  1. **초기화**:
     - `mdist`: 모든 값을 무한대(`float("inf")`)로 초기화합니다. 출발 정점 `src`는 0으로 설정합니다.
     - `vset`: 모든 값을 `False`로 초기화합니다.
  2. **반복 (V-1회)**:
     - `minDist` 함수를 사용하여 방문하지 않은 정점 중 거리가 가장 짧은 정점 `u`를 선택합니다.
     - `u`를 방문 처리(`vset[u] = True`)합니다.
     - `u`와 인접한 모든 정점 `v`에 대해 거리 갱신(Relaxation)을 수행합니다.
       - 조건: `v`가 방문되지 않았고, `u`와 `v` 사이에 간선이 있으며, `u`를 거쳐 `v`로 가는 거리가 기존 `v`까지의 거리보다 짧을 경우.
       - 갱신: `mdist[v] = mdist[u] + graph[u][v]`
  3. **출력**: `printDist` 함수를 호출하여 결과를 출력합니다.

### `printDist(dist, V)`

- **목적**: 계산된 최단 거리를 출력합니다. 도달할 수 없는 경우 "INF"로 표시합니다.

`if __name__ == "__main__":` 블록을 통해 스크립트를 직접 실행할 수 있습니다. 사용자로부터 정점 수, 간선 수, 간선 정보, 출발 정점을 입력받습니다.


## 참고 사항
- 이 구현은 인접 행렬을 사용하므로 정점의 개수가 $V$일 때 시간 복잡도는 $O(V^2)$입니다.
- 간선이 많은 밀집 그래프(Dense Graph)에서는 효율적일 수 있으나, 희소 그래프(Sparse Graph)에서는 인접 리스트와 우선순위 큐를 사용하는 방식($O(E \log V)$)이 더 효율적입니다.
