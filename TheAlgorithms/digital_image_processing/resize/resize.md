# 최근접 이웃 보간법 (Nearest Neighbour Interpolation)

이 문서는 `resize.py` 파일에 구현된 **최근접 이웃 보간법(Nearest Neighbour Interpolation)** 알고리즘에 대해 설명합니다. 이 방법은 이미지를 확대하거나 축소할 때 가장 간단하고 빠른 방식 중 하나입니다.

## 개요

이미지의 크기를 변경할 때, 새로운 픽셀 위치에 어떤 색상 값을 채워야 할지 결정해야 합니다. 최근접 이웃 알고리즘은 결과 이미지의 픽셀 좌표를 원본 이미지의 좌표로 역매핑했을 때, 가장 가까운 정수 좌표의 픽셀 값을 그대로 가져와 사용합니다.

## 클래스: `NearestNeighbour`

### `__init__(self, img, dst_width: int, dst_height: int)`
- **목적**: 리사이징 작업을 위한 초기 설정을 수행합니다.
- **매개변수**:
  - `img`: 원본 이미지 (NumPy 배열).
  - `dst_width`, `dst_height`: 변경하려는 목표 너비와 높이.
- **주요 동작**:
  - 원본 이미지의 크기(`src_w`, `src_h`)를 확인합니다.
  - 비율(`ratio_x`, `ratio_y`)을 계산합니다. (원본 크기 / 목표 크기)
  - 결과 이미지를 저장할 배열(`output`)을 생성합니다.

### `process(self)`
- **목적**: 리사이징 로직을 실행합니다.
- **동작**:
  - 목표 이미지의 모든 픽셀(`i`, `j`)을 순회합니다.
  - `get_y(i)`와 `get_x(j)`를 통해 원본 이미지에서 대응되는 좌표를 찾습니다.
  - 원본 좌표의 픽셀 값을 목표 이미지의 현재 위치에 복사합니다.

### `get_x(self, x: int) -> int` / `get_y(self, y: int) -> int`
- **목적**: 목표 이미지의 좌표를 원본 이미지의 좌표로 변환합니다.
- **공식**: `int(ratio * coordinate)`
  - 단순히 비율을 곱한 뒤 정수형으로 변환(버림)하여 가장 가까운 원본 픽셀 인덱스를 구합니다.

## 사용법

`if __name__ == "__main__":` 블록에서 사용 예시를 확인할 수 있습니다:

1. `image_data/lena.jpg` 이미지를 불러옵니다.
2. 목표 크기를 800x600으로 설정합니다.
3. `NearestNeighbour` 인스턴스를 생성하고 `process()`를 호출합니다.
4. 원본 크기에서 변경된 크기로 리사이징된 이미지를 화면에 표시합니다.
