# 이미지 반전 (Convert to Negative)

이 문서는 `convert_to_negative.py` 파일에 구현된 이미지 반전 알고리즘에 대해 설명합니다. 이 코드는 OpenCV를 사용하여 컬러 이미지를 네거티브(반전) 이미지로 변환합니다.

## 개요

이미지의 각 픽셀 색상 값을 반전시킵니다. RGB 모델에서 반전된 색상은 최대 값(255)에서 현재 픽셀 값을 뺀 결과입니다.

## 주요 함수: `convert_to_negative`

### `convert_to_negative(img)`
- **목적**: 입력 이미지의 모든 픽셀을 순회하며 색상을 반전시킵니다.
- **매개변수**:
  - `img`: OpenCV로 읽어온 이미지 객체 (NumPy 배열).
- **동작 원리**:
  1. 이미지의 높이(`pixel_h`)와 너비(`pixel_v`)를 구합니다.
  2. 이중 `for` 루프를 사용하여 모든 픽셀(`i`, `j`)을 방문합니다.
  3. 각 픽셀의 RGB 값에 대해 `[255, 255, 255] - 현재 픽셀 값` 연산을 수행합니다.
  4. 계산된 반전 값을 해당 픽셀에 다시 할당합니다.

## 사용법

`if __name__ == "__main__":` 블록에서 사용 예시를 확인할 수 있습니다:

1. `image_data/lena.jpg` 이미지를 컬러 모드(`1`)로 읽어옵니다.
2. `convert_to_negative` 함수를 호출하여 이미지를 변환합니다.
3. `imshow`를 사용하여 결과 이미지를 화면에 표시합니다.
4. 아무 키나 누르면 창이 닫힙니다.

## 참고 사항

- **성능**: 이 코드는 Python의 이중 반복문을 사용하여 픽셀을 하나씩 처리하므로, 해상도가 높은 이미지에서는 처리 속도가 느릴 수 있습니다. 실무에서는 NumPy의 벡터화 연산(예: `255 - img`)이나 OpenCV의 내장 함수(`cv2.bitwise_not`)를 사용하는 것이 훨씬 빠릅니다.
