# 룬 알고리즘 (Luhn Algorithm)

이 문서는 `luhn.py` 파일에 구현된 **룬 알고리즘(Luhn Algorithm)**에 대해 설명합니다.

## 개요

룬 알고리즘(Luhn Algorithm)은 신용카드 번호, IMEI 번호 등 다양한 식별 번호의 유효성을 검증하기 위해 널리 사용되는 간단한 체크섬 공식입니다. 1954년 한스 피터 룬(Hans Peter Luhn)이 개발했습니다.

이 알고리즘은 입력 오류나 우발적인 숫자 변경을 감지하는 데 유용하지만, 악의적인 공격을 방어하기 위한 암호화 해시 함수는 아닙니다.

## 주요 함수: `is_luhn`

### `is_luhn(string: str) -> bool`

- **목적**: 입력된 숫자 문자열이 룬 알고리즘을 만족하는지 검증합니다.
- **매개변수**:
  - `string`: 검증할 숫자 문자열 (예: "79927398713").
- **반환값**: 유효하면 `True`, 그렇지 않으면 `False`.

### 알고리즘 동작 원리

1. **분리**: 입력 문자열의 마지막 숫자를 **체크 디지트(Check Digit)**로 분리하고, 나머지 앞부분을 검증할 숫자로 둡니다.
2. **역순 정렬**: 나머지 숫자들을 역순으로 뒤집습니다.
3. **반복 연산**:
   - 역순으로 정렬된 리스트를 순회합니다.
   - **짝수 인덱스 (0, 2, 4...)**: 원래 문자열 기준으로는 오른쪽 끝에서 두 번째 숫자부터 시작하여 하나 건너 하나씩 해당합니다. 이 숫자들을 **2배**로 만듭니다.
     - 만약 2배 한 값이 9보다 크다면, 9를 뺍니다 (예: $8 \times 2 = 16 \rightarrow 16 - 9 = 7$). 이는 각 자릿수를 더하는 것과 동일한 효과입니다 ($1 + 6 = 7$).
   - **홀수 인덱스 (1, 3, 5...)**: 숫자를 그대로 둡니다.
4. **합계 계산**: 위 과정에서 계산된 모든 값들과 처음에 분리해 둔 체크 디지트를 모두 더합니다.
5. **검증**: 총합이 10으로 나누어떨어지면(`sum % 10 == 0`) 유효한 번호입니다.

## 사용법

`if __name__ == "__main__":` 블록에서 `doctest`를 실행하여 코드를 검증하고 예제를 실행합니다.

```python
# 유효한 케이스
assert is_luhn("79927398713") == True

# 유효하지 않은 케이스
assert is_luhn("79927398714") == False
```
