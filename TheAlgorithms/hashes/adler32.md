# Adler-32 체크섬 알고리즘 (Adler-32 Checksum Algorithm)

이 문서는 `adler32.py` 파일에 구현된 **Adler-32** 체크섬 알고리즘에 대해 설명합니다.

## 개요

Adler-32는 1995년 Mark Adler가 고안한 체크섬 알고리즘입니다. 같은 길이의 순환 중복 검사(CRC)와 비교할 때, 신뢰성을 조금 희생하는 대신 **속도**를 우선시하도록 설계되었습니다. Fletcher-16보다는 신뢰성이 높고 Fletcher-32보다는 약간 낮습니다.

## 주요 함수: `adler32`

### `adler32(plain_text: str) -> int`

- **목적**: 주어진 문자열에 대한 Adler-32 체크섬 값을 계산합니다.
- **매개변수**:
  - `plain_text`: 체크섬을 계산할 입력 문자열.
- **반환값**: 계산된 Adler-32 해시 값 (정수).

### 알고리즘 동작 원리

Adler-32는 두 개의 16비트 체크섬 `A`와 `B`를 계산하여 이를 32비트 정수로 결합합니다.

1. **상수 정의**:
   - `MOD_ADLER`: 65521 (65536보다 작은 가장 큰 소수).

2. **초기화**:
   - `a`: 1로 초기화됩니다.
   - `b`: 0으로 초기화됩니다.

3. **반복 계산**:
   - 입력 문자열의 각 문자에 대해 다음을 수행합니다:
     - `a`: 현재 문자의 ASCII 값을 더하고 `MOD_ADLER`로 나눈 나머지를 구합니다.
     - `b`: 현재의 `a` 값을 더하고 `MOD_ADLER`로 나눈 나머지를 구합니다.

4. **결과 반환**:
   - `b`를 상위 16비트로, `a`를 하위 16비트로 배치하여 결합합니다.
   - 수식: `(b << 16) | a`

## 사용법

`doctest`에 포함된 예제를 통해 동작을 확인할 수 있습니다.

```python
adler32('Algorithms')
# 결과: 363791387

adler32('go adler em all')
# 결과: 708642122
```
