# SDBM 해시 알고리즘 (SDBM Hash Algorithm)

이 문서는 `sdbm.py` 파일에 구현된 **SDBM** 해시 알고리즘에 대해 설명합니다.

## 개요

SDBM(Simple Database Manager) 해시는 원래 `sdbm` 데이터베이스 라이브러리를 위해 만들어진 비암호화 해시 함수입니다. 이 알고리즘은 비트를 효과적으로 섞어(Scrambling) 키의 분포를 좋게 만드는 데 탁월한 성능을 보여, 해시 테이블 등에서 충돌을 줄이는 데 유용합니다.

알고리즘의 기본 원리는 다음과 같은 의사 코드로 표현할 수 있습니다.
`hash(i) = hash(i - 1) * 65599 + str[i]`

이 구현에서는 곱셈 대신 비트 시프트(Bit Shift) 연산을 사용하여 더 빠른 버전으로 작성되었습니다.

## 주요 함수: `sdbm`

### `sdbm(plain_text: str) -> int`

- **목적**: 주어진 문자열에 대한 SDBM 해시 값을 계산합니다.
- **매개변수**:
  - `plain_text`: 해시할 입력 문자열.
- **반환값**: 계산된 해시 값 (정수).

### 알고리즘 동작 원리

1.  `hash` 변수를 `0`으로 초기화합니다.
2.  입력 문자열의 각 문자를 순회합니다.
3.  각 문자에 대해 `hash = ord(plain_chr) + (hash << 6) + (hash << 16) - hash` 공식을 적용하여 `hash` 값을 갱신합니다.
    - `(hash << 6)`는 `hash * 64`와 같습니다.
    - `(hash << 16)`는 `hash * 65536`과 같습니다.
    - 따라서 위 식은 `ord(c) + hash * (64 + 65536 - 1)` 즉, `ord(c) + hash * 65599`와 유사한 효과를 냅니다.
4.  모든 문자 처리가 끝나면 최종 `hash` 값을 반환합니다.

## 사용법

`doctest`에 포함된 예제를 통해 동작을 확인할 수 있습니다.

```python
sdbm('Algorithms')
# 결과: 1462174910723540325254304520539387479031000036

sdbm('scramble bits')
# 결과: 730247649148944819640658295400555317318720608290373040936089
```
