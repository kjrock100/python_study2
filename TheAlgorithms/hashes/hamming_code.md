# 해밍 코드 (Hamming Code)

이 문서는 `hamming_code.py` 파일에 구현된 **해밍 코드(Hamming Code)** 알고리즘에 대해 설명합니다.

## 개요

해밍 코드는 통신 분야에서 널리 사용되는 선형 오류 정정 코드(Linear Error-Correcting Code)의 일종입니다. 데이터 전송 시 오류를 감지하고 수정할 수 있도록 데이터 비트에 **패리티 비트(Parity Bit)**를 추가하는 방식입니다.

이 구현은 메시지를 인코딩하는 송신자(Emitter)와 메시지를 디코딩하고 무결성을 확인하는 수신자(Receptor)의 기능을 제공합니다.

## 주요 함수

### 이진 변환 함수
- **`text_to_bits(text)`**: 문자열 텍스트를 이진수 문자열로 변환합니다.
- **`text_from_bits(bits)`**: 이진수 문자열을 다시 원래의 텍스트로 변환합니다.

### 해밍 코드 함수

#### `emitterConverter(sizePar, data)`
- **목적**: 데이터를 전송하기 위해 해밍 코드로 인코딩합니다.
- **매개변수**:
  - `sizePar`: 추가할 패리티 비트의 수.
  - `data`: 전송할 데이터 비트 문자열.
- **동작 원리**:
  1. 2의 거듭제곱 위치($1, 2, 4, 8, \dots$)에 패리티 비트가 들어갈 자리를 비워두고 데이터를 배치합니다.
  2. 각 패리티 비트가 담당하는 데이터 비트들을 확인하여 패리티 값을 계산합니다.
  3. 계산된 패리티 비트를 해당 위치에 채워 넣어 최종 전송 메시지를 생성합니다.

#### `receptorConverter(sizePar, data)`
- **목적**: 수신된 메시지를 디코딩하고 데이터 무결성을 확인합니다.
- **매개변수**:
  - `sizePar`: 메시지에 포함된 패리티 비트의 수.
  - `data`: 수신된 해밍 코드 메시지.
- **동작 원리**:
  1. 수신된 메시지에서 데이터 비트와 패리티 비트를 분리합니다.
  2. 추출한 데이터 비트를 사용하여 패리티 비트를 다시 계산합니다.
  3. 수신된 패리티 비트와 재계산된 패리티 비트를 비교합니다.
  4. 두 값이 일치하면 데이터 무결성이 확인된 것이며(`True`), 그렇지 않으면 오류가 발생한 것입니다(`False`).
- **반환값**: `(디코딩된 데이터, 무결성 여부)` 튜플.

## 사용법

코드 하단의 주석 처리된 예제(`Example how to use`)를 참고하여 사용할 수 있습니다.

1. **설정**: 패리티 비트 수(`sizePari`)를 설정합니다.
2. **변환**: 전송할 텍스트를 `text_to_bits`를 통해 이진수로 변환합니다.
3. **인코딩**: `emitterConverter`를 사용하여 이진 데이터를 해밍 코드로 변환합니다.
4. **전송 및 오류 시뮬레이션**: 전송된 데이터의 특정 비트를 임의로 변경하여 오류를 시뮬레이션할 수 있습니다.
5. **디코딩**: `receptorConverter`를 사용하여 데이터를 복원하고 `ack` 변수를 통해 데이터가 손상되었는지 확인합니다.

```python
# 예시
sizePari = 4
text = "Message01"
binaryText = text_to_bits(text)

# 인코딩
dataOut = emitterConverter(sizePari, binaryText)

# 디코딩
dataReceiv, ack = receptorConverter(sizePari, dataOut)
print(f"Data integrity: {ack}")
```

## 요구 사항
