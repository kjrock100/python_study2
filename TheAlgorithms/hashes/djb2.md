# djb2 해시 알고리즘 (djb2 Hash Algorithm)

이 문서는 `djb2.py` 파일에 구현된 **djb2** 해시 알고리즘에 대해 설명합니다.

## 개요

djb2는 Dan Bernstein이 고안한 간단하면서도 효율적인 비암호화 해시 함수입니다. 이 알고리즘은 비트를 효과적으로 섞어 키의 분포를 좋게 만드는 것으로 알려져 있으며, 구현이 간단하고 속도가 빨라 널리 사용됩니다.

알고리즘은 두 개의 "마법의 숫자(magic number)"를 사용합니다.
- **5381**: 해시 계산을 시작하는 초기값.
- **33**: 각 문자를 처리할 때 이전 해시 값에 곱해지는 승수.

## 주요 함수: `djb2`

### `djb2(s: str) -> int`

- **목적**: 주어진 문자열 `s`에 대한 djb2 해시 값을 계산합니다.
- **매개변수**:
  - `s`: 해시할 입력 문자열.
- **반환값**: 계산된 32비트 해시 값 (정수).

### 알고리즘 동작 원리

1.  `hash` 변수를 초기값 `5381`로 설정합니다.
2.  입력 문자열의 각 문자를 순회합니다.
3.  각 문자에 대해 `hash = ((hash << 5) + hash) + ord(x)` 공식을 적용하여 `hash` 값을 갱신합니다.
    - `(hash << 5) + hash`는 `hash * 32 + hash`, 즉 `hash * 33`과 동일한 연산입니다. 비트 시프트(Bit Shift) 연산을 사용하면 곱셈보다 더 빠르게 계산할 수 있습니다.
4.  모든 문자에 대한 계산이 끝나면, `& 0xFFFFFFFF` 연산을 통해 결과를 32비트 부호 없는 정수 범위로 제한하여 반환합니다.

## 사용법

`doctest`에 포함된 예제를 통해 동작을 확인할 수 있습니다.

```python
djb2('Algorithms')
# 결과: 3782405311

djb2('scramble bits')
# 결과: 1609059040
```
