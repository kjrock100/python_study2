# 코크 눈송이 (Koch Snowflake)

이 문서는 `koch_snowflake.py` 파일에 구현된 **코크 눈송이(Koch Snowflake)** 생성 및 시각화 알고리즘에 대해 설명합니다.

## 개요

코크 눈송이는 가장 먼저 설명된 프랙탈 곡선 중 하나입니다. 정삼각형에서 시작하여 각 변에 대해 반복적으로 작은 정삼각형을 추가하는 방식으로 만들어집니다.

## 알고리즘 동작 원리

각 단계마다 모든 선분에 대해 다음 과정을 수행합니다:
1. 선분을 3등분합니다.
2. 가운데 선분을 밑변으로 하는 정삼각형을 바깥쪽으로 그립니다.
3. 원래의 가운데 선분을 제거합니다.

이 과정을 반복하면 눈송이 모양의 프랙탈 구조가 형성됩니다.

## 주요 함수

### `iterate(initial_vectors, steps)`
- **목적**: 초기 벡터(점)들로부터 시작하여 지정된 횟수(`steps`)만큼 반복 과정을 수행합니다.
- **주의**: 단계가 높아질수록 계산량이 기하급수적으로 늘어나므로 5 이상의 값은 주의해야 합니다.

### `iteration_step(vectors)`
- **목적**: 한 단계의 반복을 수행합니다.
- **동작**: 인접한 두 벡터(선분) 사이를 4개의 새로운 선분으로 나눕니다.
  - 시작점 -> 1/3 지점
  - 1/3 지점 -> 튀어나온 삼각형의 꼭짓점 (60도 회전)
  - 튀어나온 삼각형의 꼭짓점 -> 2/3 지점
  - 2/3 지점 -> 끝점

### `rotate(vector, angle_in_degrees)`
- **목적**: 2D 벡터를 주어진 각도만큼 회전시킵니다. 회전 행렬을 사용합니다.

### `plot(vectors)`
- **목적**: `matplotlib`를 사용하여 계산된 벡터들을 시각화합니다.
- **특징**: 그래프의 비율을 `equal`로 설정하여 왜곡 없이 눈송이 모양을 보여줍니다.

## 사용법

`if __name__ == "__main__":` 블록에서 실행 예시를 확인할 수 있습니다.

```python
# 초기 정삼각형 정의
INITIAL_VECTORS = [VECTOR_1, VECTOR_2, VECTOR_3, VECTOR_1]

# 5단계 반복 후 플롯
processed_vectors = iterate(INITIAL_VECTORS, 5)
plot(processed_vectors)
```
