# 시어핀스키 삼각형 (Sierpinski Triangle)

이 문서는 `sierpinski_triangle.py` 파일에 구현된 **시어핀스키 삼각형** 생성 알고리즘에 대해 설명합니다.

## 개요

시어핀스키 삼각형은 전체적인 모양이 정삼각형이며, 내부가 재귀적으로 더 작은 정삼각형들로 분할되는 프랙탈 도형입니다. 이 코드는 Python의 `turtle` 그래픽 라이브러리를 사용하여 이를 시각화합니다.

## 주요 함수

### `getMid(p1, p2)`
- **목적**: 두 점 `p1`과 `p2`의 중점(Midpoint)을 계산합니다.
- **매개변수**: `p1`, `p2`는 각각 (x, y) 좌표를 나타냅니다.
- **반환값**: 두 점의 중점 좌표.

### `triangle(points, depth)`
- **목적**: 주어진 깊이(`depth`)만큼 재귀적으로 삼각형을 그립니다.
- **매개변수**:
  - `points`: 삼각형의 세 꼭짓점 좌표가 담긴 리스트 `[[x1, y1], [x2, y2], [x3, y3]]`.
  - `depth`: 프랙탈의 재귀 깊이.
- **알고리즘 동작**:
  1. `turtle` 펜을 사용하여 주어진 세 점을 잇는 삼각형을 그립니다.
  2. `depth`가 0보다 크면, 현재 삼각형의 각 꼭짓점과 그 대변의 중점들을 이용하여 3개의 작은 삼각형에 대해 재귀적으로 `triangle` 함수를 호출합니다 (`depth - 1`).

## 사용법

이 스크립트는 커맨드 라인 인자로 재귀 깊이(정수)를 받습니다.

```bash
python sierpinski_triangle.py <depth>
```

**실행 예시**:
```bash
python sierpinski_triangle.py 3
```
위 명령어를 실행하면 깊이가 3인 시어핀스키 삼각형이 그려진 윈도우가 열립니다.

## 요구 사항
