# `a1z26.py` 코드 설명

이 문서는 `a1z26.py` 파이썬 스크립트에 포함된 함수들을 설명합니다. 이 스크립트는 **A1Z26 암호**를 구현하여, 문자열을 알파벳 순서에 해당하는 숫자 리스트로 변환하거나 그 반대의 작업을 수행합니다.

## 목차
1.  A1Z26 암호란?
2.  함수 설명
    -   `encode(plain)`
    -   `decode(encoded)`
    -   `main()`
3.  실행 방법
4.  코드 개선 제안

## A1Z26 암호란?

A1Z26은 가장 간단한 형태의 대체 암호 중 하나입니다. 각 알파벳 문자를 알파벳에서의 위치에 해당하는 숫자로 바꿉니다.

-   `A`는 `1`, `B`는 `2`, `C`는 `3`, ..., `Z`는 `26`으로 변환됩니다.
-   이 스크립트는 소문자를 기준으로 동작합니다.

## 함수 설명

### `encode(plain: str) -> list[int]`

알파벳 문자열을 A1Z26 규칙에 따라 숫자 리스트로 **인코딩(암호화)**합니다.

-   **알고리즘**:
    -   입력된 문자열(`plain`)의 각 문자에 대해 `ord()` 함수를 사용하여 ASCII 코드 값을 얻습니다.
    -   소문자 'a'의 ASCII 값은 97입니다. 따라서 각 문자의 ASCII 값에서 96을 빼면 알파벳 순서(a=1, b=2, ...)를 얻을 수 있습니다.
    -   `ord(elem) - 96`

```python
>>> encode("hello")
[8, 5, 12, 12, 15]
```

### `decode(encoded: list[int]) -> str`

숫자 리스트를 A1Z26 규칙에 따라 다시 알파벳 문자열로 **디코딩(복호화)**합니다.

-   **알고리즘**:
    -   입력된 숫자 리스트(`encoded`)의 각 숫자에 96을 더하여 해당 알파벳의 ASCII 코드 값을 만듭니다.
    -   `chr()` 함수를 사용하여 이 ASCII 코드 값을 다시 문자로 변환합니다.
    -   `chr(elem + 96)`

```python
>>> decode([8, 5, 12, 12, 15])
'hello'
```

### `main()`

사용자로부터 문자열을 입력받아 인코딩 및 디코딩 과정을 시연합니다.

-   **동작**:
    1.  사용자에게 입력을 요청합니다. (`-> `)
    2.  입력된 문자열의 양 끝 공백을 제거하고(`strip()`) 모두 소문자로 변환합니다(`lower()`).
    3.  `encode()` 함수를 호출하여 결과를 출력합니다.
    4.  인코딩된 결과를 다시 `decode()` 함수에 전달하여 원본 문자열로 복원되는 것을 보여줍니다.

## 실행 방법

스크립트를 직접 실행하면 사용자 입력을 받아 암호화 및 복호화를 수행합니다.

```bash
python a1z26.py
```

실행 후, 터미널에 변환할 문자열을 입력합니다.

```
-> Hello World
Encoded:  [8, 5, 12, 12, 15, 23, 15, 18, 12, 4]
Decoded: helloworld
```

## 코드 개선 제안

1.  **비-알파벳 문자 처리**: 현재 코드는 입력에 알파벳 외의 문자(공백, 숫자, 특수문자)가 포함될 경우 의도치 않은 결과를 내거나, `main` 함수에서 처리되지 않은 경우 오류를 발생시킬 수 있습니다. `encode` 함수 내에서 `isalpha()`를 사용하여 알파벳인 경우에만 변환하도록 수정하면 코드가 더 안정적으로 동작합니다.

2.  **'매직 넘버' 제거**: 코드에 사용된 숫자 `96`은 '매직 넘버'로, 그 의미를 즉시 파악하기 어렵습니다. `ord('a') - 1` 과 같이 계산하여 사용하면 코드의 가독성이 크게 향상됩니다.

    ```python
    # 개선 제안 예시
    def encode(plain: str) -> list[int]:
        offset = ord('a') - 1
        return [ord(char) - offset for char in plain.lower() if char.isalpha()]

    def decode(encoded: list[int]) -> str:
        offset = ord('a') - 1
        return "".join(chr(num + offset) for num in encoded)
    ```
