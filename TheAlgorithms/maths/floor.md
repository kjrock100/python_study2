# Floor (내림) 알고리즘

이 문서는 `floor.py` 파일에 구현된 **내림(Floor)** 함수에 대한 설명입니다.

## 개요

**내림 함수**는 실수 `x`를 입력받아 `x`보다 작거나 같은 정수 중 가장 큰 정수를 반환하는 함수입니다. 수학적으로 $\lfloor x \rfloor$로 표기하며, 가우스 기호 $[x]$와 동일한 의미를 가집니다.

## 함수 설명

### `floor(x) -> int`

주어진 숫자 `x`의 내림 값을 계산하여 정수 형태로 반환합니다.

#### 매개변수 (Parameters)

- `x`: 내림을 계산할 숫자 (정수 또는 실수).

#### 알고리즘 (Algorithm)

파이썬의 내장 함수 `int()`는 소수점 이하를 0 방향으로 버리는(truncate) 특성이 있습니다. 이를 이용하여 다음과 같이 구현되었습니다.

1. `int(x)`를 통해 `x`의 정수 부분을 구합니다.
2. `x - int(x)` 값을 확인합니다.
   - **0 이상인 경우**: `x`가 양수이거나 정수인 경우입니다. 이때는 `int(x)`가 곧 내림 값이므로 그대로 반환합니다.
     - 예: `floor(1.1)` -> `1`, `floor(2.0)` -> `2`
   - **0 미만인 경우**: `x`가 음수이면서 소수 부분이 있는 경우입니다. 이때는 `int(x)` 값보다 1 작은 정수가 내림 값이므로 `int(x) - 1`을 반환합니다.
     - 예: `floor(-1.1)` -> `-1 - 1` -> `-2`
