# Area Under Curve (곡선 아래 면적) 알고리즘

이 문서는 `area_under_curve.py` 파일에 구현된 **사다리꼴 공식(Trapezoidal Rule)**을 이용한 곡선 아래 면적 근사 알고리즘에 대한 설명입니다.

## 개요

주어진 함수(곡선)와 구간에 대해, 구간을 잘게 나누어 사다리꼴들의 넓이 합으로 곡선과 x축 사이의 면적을 근사 계산합니다. 적분(Integral)의 개념을 수치적으로 구현한 방법 중 하나입니다.

## 함수 설명

### `trapezoidal_area(fnc, x_start, x_end, steps=100) -> float`

사다리꼴 공식을 사용하여 주어진 구간에서 함수 `fnc`의 그래프 아래 면적을 계산합니다.

#### 매개변수 (Parameters)

- `fnc` (`Callable[[int | float], int | float]`): 곡선을 정의하는 함수입니다. $y = f(x)$ 형태의 함수로, x값을 입력받아 y값을 반환해야 합니다.
- `x_start` (`int` | `float`): 적분 구간의 시작점(왼쪽 끝)입니다.
- `x_end` (`int` | `float`): 적분 구간의 끝점(오른쪽 끝)입니다.
- `steps` (`int`): 구간을 나눌 등분(사다리꼴)의 개수입니다. 값이 클수록 근사값이 실제 면적에 가까워집니다. (기본값: 100)

#### 알고리즘 (Algorithm)

1. 시작점 `x1`을 `x_start`로 설정하고 해당 지점의 함수값 `fx1`을 계산합니다.
2. `steps` 횟수만큼 반복하며 다음을 수행합니다:
   - 현재 구간의 끝점 `x2`를 계산합니다. (전체 구간 길이를 `steps`로 나눈 간격만큼 이동)
   - `x2`에서의 함수값 `fx2`를 계산합니다.
   - 두 지점(`x1`, `x2`) 사이의 사다리꼴 넓이를 계산하여 총 면적 `area`에 더합니다.
     - 계산 식: `abs(fx2 + fx1) * (x2 - x1) / 2`
     - **참고**: 코드에서는 `abs()`를 사용하여 함수값 합의 절대값을 취하므로, x축 아래에 있는 영역도 양수의 면적(기하학적 넓이)으로 합산됩니다.
   - 다음 반복을 위해 `x1`, `fx1`을 현재의 `x2`, `fx2`로 갱신합니다.
3. 누적된 총 면적 `area`를 반환합니다.

## 실행 예시

파일을 직접 실행하면(`if __name__ == "__main__":`), 함수 $f(x) = x^3 + x^2$에 대해 $x = -5$에서 $x = 5$까지의 면적을 단계 수(`steps`)를 10부터 100,000까지 늘려가며 계산하고 결과를 출력합니다.

```python
def f(x):
    return x**3 + x**2

# steps=1000 일 때의 면적 계산
area = trapezoidal_area(f, -5, 5, 1000)
print(f"Calculated area: {area}")
```
