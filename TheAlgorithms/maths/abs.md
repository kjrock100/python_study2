# Absolute Value (절대값) 알고리즘

이 문서는 `abs.py` 파일에 구현된 **절대값 계산** 알고리즘에 대한 설명입니다.

## 개요

주어진 숫자의 절대값을 계산하여 반환하는 기능을 제공합니다. 파이썬의 내장 함수 `abs()`와 유사한 기능을 직접 구현한 것입니다.

## 함수 설명

### `abs_val(num)`

입력된 숫자의 절대값을 반환합니다.

#### 매개변수 (Parameters)

- `num`: 절대값을 구할 숫자 (정수 `int` 또는 실수 `float`).

#### 알고리즘 (Algorithm)

1. 입력된 숫자 `num`이 0보다 작은지 확인합니다.
2. 0보다 작으면 `-num` (부호를 반전)을 반환합니다.
3. 그렇지 않으면(0 이상이면) `num` 그대로 반환합니다.

### `test_abs_val()`

`abs_val` 함수가 올바르게 작동하는지 검증하는 테스트 함수입니다.

- `assert` 문을 사용하여 0, 양수, 큰 음수에 대한 결과가 예상값과 일치하는지 확인합니다.

## 실행 예시

파일을 직접 실행하면(`if __name__ == "__main__":`), 숫자 -34에 대한 절대값을 계산하고 결과를 출력합니다.

```python
if __name__ == "__main__":
    print(abs_val(-34))  # --> 34
```
