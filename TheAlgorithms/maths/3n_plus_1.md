# 3n + 1 (Collatz Conjecture) 알고리즘

이 문서는 `3n_plus_1.py` 파일에 구현된 **3n + 1 문제** (또는 **콜라츠 추측**) 알고리즘에 대한 설명입니다.

## 개요

**콜라츠 추측(Collatz conjecture)**은 어떤 자연수에서 시작하더라도 아래의 규칙을 반복하면 결국 1에 도달한다는 수학적 추측입니다.

1. 숫자가 짝수라면 2로 나눕니다.
2. 숫자가 홀수라면 3을 곱하고 1을 더합니다.
3. 결과가 1이 될 때까지 이 과정을 반복합니다.

## 함수 설명

### `n31(a: int) -> tuple[list[int], int]`

주어진 양의 정수 `a`에 대해 콜라츠 수열과 그 길이를 계산하여 반환합니다.

#### 매개변수 (Parameters)

- `a` (`int`): 수열을 시작할 양의 정수입니다.

#### 예외 처리 (Error Handling)

1. **TypeError**: 입력값 `a`가 정수(`int`)가 아닌 경우 발생합니다.
2. **ValueError**: 입력값 `a`가 1 미만인 경우 발생합니다. (코드상의 에러 메시지는 "greater than 1"이라고 명시되어 있으나, 실제 조건은 `a < 1`일 때 에러가 발생하므로 1 이상의 정수는 허용됩니다.)

#### 알고리즘 (Algorithm)

1. 입력받은 수 `a`를 리스트 `path`에 저장합니다.
2. `a`가 1이 될 때까지 다음을 반복합니다:
   - `a`가 짝수이면 2로 나눕니다 (`//` 연산자 사용).
   - `a`가 홀수이면 3을 곱하고 1을 더합니다.
   - 변경된 `a`를 `path`에 추가합니다.
3. 최종적으로 생성된 수열 `path`와 그 길이 `len(path)`를 튜플 형태로 반환합니다.

### `test_n31()`

`n31` 함수가 올바르게 작동하는지 검증하는 테스트 함수입니다.

- `assert` 문을 사용하여 입력값 4, 11, 31에 대한 결과가 예상값과 일치하는지 확인합니다.

## 실행 예시

파일을 직접 실행하면(`if __name__ == "__main__":`), 숫자 4에 대한 콜라츠 수열을 계산하고 결과를 출력합니다.

```python
num = 4
path, length = n31(num)
print(f"The Collatz sequence of {num} took {length} steps. \nPath: {path}")
```

**출력 결과:**

```
The Collatz sequence of 4 took 3 steps.
Path: [4, 2, 1]
```
