# Line Length (Arc Length) 알고리즘

이 문서는 `line_length.py` 파일에 구현된 **곡선의 길이(Arc Length)** 근사 알고리즘에 대한 설명입니다.

## 개요

주어진 함수 $f(x)$가 정의하는 곡선에 대해, 구간 $[x_{start}, x_{end}]$에서의 길이를 근사적으로 계산합니다. 곡선을 작은 선분들의 연속으로 취급하여 각 선분의 길이를 합산하는 방식을 사용합니다.

## 함수 설명

### `line_length(fnc, x_start, x_end, steps=100) -> float`

주어진 구간에서 함수의 그래프를 따라 곡선의 길이를 계산하여 반환합니다.

#### 매개변수 (Parameters)

- `fnc` (`Callable`): 곡선을 정의하는 함수입니다. $x$를 입력받아 $y$를 반환해야 합니다.
- `x_start` (`int` | `float`): 구간의 시작점(왼쪽 끝)입니다.
- `x_end` (`int` | `float`): 구간의 끝점(오른쪽 끝)입니다.
- `steps` (`int`): 구간을 나눌 선분의 개수입니다. 값이 클수록 정확도가 높아집니다. (기본값: 100)

#### 알고리즘 (Algorithm)

1. 시작점 `x1`을 `x_start`로, `fx1`을 `fnc(x_start)`로 설정합니다.
2. `steps` 횟수만큼 반복하며 다음을 수행합니다:
   - 다음 지점 `x2`를 계산합니다. (전체 구간 길이를 `steps`로 나눈 간격만큼 이동)
   - `x2`에서의 함수값 `fx2`를 계산합니다.
   - 두 점 $(x_1, f(x_1))$과 $(x_2, f(x_2))$ 사이의 거리를 `math.hypot`을 사용하여 계산하고 총 길이 `length`에 더합니다.
     - 거리 공식: $\sqrt{(x_2 - x_1)^2 + (f(x_2) - f(x_1))^2}$
   - 다음 반복을 위해 `x1`, `fx1`을 현재의 `x2`, `fx2`로 갱신합니다.
3. 누적된 총 길이 `length`를 반환합니다.

## 실행 예시

파일을 직접 실행하면(`if __name__ == "__main__":`), 함수 $f(x) = \sin(10x)$에 대해 $x = -10$에서 $x = 10$까지의 길이를 단계 수(`steps`)를 늘려가며 계산하고 결과를 출력합니다.

```python
def f(x):
    return math.sin(10 * x)

# ... (반복문 실행)
```

**출력 예시:**

```
f(x) = sin(10 * x)
The length of the curve from x = -10 to x = 10 is:
With 10 steps: 20.0
With 100 steps: 129.53...
```
