# Add (덧셈) 알고리즘

이 문서는 `add.py` 파일에 구현된 **덧셈** 알고리즘에 대한 설명입니다.

## 개요

두 개의 숫자를 입력받아 그 합을 반환하는 간단한 함수입니다.

## 함수 설명

### `add(a, b)`

두 수의 합을 계산하여 반환합니다.

#### 매개변수 (Parameters)

- `a`: 첫 번째 숫자 (정수 또는 실수 등 덧셈이 가능한 타입).
- `b`: 두 번째 숫자.

#### 알고리즘 (Algorithm)

1. 파이썬의 `+` 연산자를 사용하여 `a`와 `b`를 더합니다.
2. 결과를 반환합니다.

## 테스트 및 실행

파일을 직접 실행하면(`if __name__ == "__main__":`), 예제 변수 `a=5`, `b=6`에 대한 덧셈 결과를 출력합니다.

```python
if __name__ == "__main__":
    a = 5
    b = 6
    print(f"The sum of {a} + {b} is {add(a, b)}")
```
