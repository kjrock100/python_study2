# Kth Lexicographic Permutation (K번째 사전순 순열) 알고리즘

이 문서는 `kth_lexicographic_permutation.py` 파일에 구현된 **K번째 사전순 순열** 찾기 알고리즘에 대한 설명입니다.

## 개요

0부터 $n-1$까지의 숫자로 이루어진 순열들을 사전순(오름차순)으로 나열했을 때, $k$번째에 해당하는 순열을 $O(n^2)$ 시간 복잡도로 찾아내는 알고리즘입니다.

예를 들어 $n=3$일 때 순열은 다음과 같습니다: 0. `[0, 1, 2]`

1. `[0, 2, 1]`
2. `[1, 0, 2]`
3. `[1, 2, 0]`
4. `[2, 0, 1]`
5. `[2, 1, 0]`

여기서 $k=3$이면 `[1, 2, 0]`을 반환합니다.

## 함수 설명

### `kthPermutation(k, n)`

$0, 1, \dots, n-1$의 순열 중 $k$번째 순열을 찾아 리스트로 반환합니다.

#### 매개변수 (Parameters)

- `k` (`int`): 찾고자 하는 순열의 인덱스 (0부터 시작).
- `n` (`int`): 원소의 개수 (0부터 $n-1$까지의 숫자 사용).

#### 예외 처리 (Error Handling)

- **AssertionError**: `k`가 가능한 순열의 총 개수($n!$) 이상이거나 음수일 경우 "k out of bounds" 에러가 발생합니다.

#### 알고리즘 (Algorithm)

이 알고리즘은 팩토리얼 진법(Factorial Number System)의 아이디어를 사용합니다.

1. **팩토리얼 계산**: $1!$부터 $(n-1)!$까지 미리 계산하여 리스트 `factorials`에 저장합니다.
2. **유효성 검사**: `k`가 범위 내에 있는지 확인합니다.
3. **순열 생성**:
   - 사용 가능한 숫자 리스트 `elements`를 `[0, 1, ..., n-1]`로 초기화합니다.
   - 가장 큰 팩토리얼부터 차례대로 `k`를 나눕니다 (`divmod`).
   - 몫(`number`)은 현재 자리에 올 숫자가 `elements` 리스트에서 몇 번째 인덱스인지를 나타냅니다.
   - 해당 숫자를 결과 리스트 `permutation`에 추가하고, `elements`에서 제거합니다.
   - 나머지(`k`)는 다음 자릿수 계산을 위해 업데이트됩니다.
4. 마지막 남은 원소를 추가하고 결과를 반환합니다.
