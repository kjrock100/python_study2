# Ceil (올림) 알고리즘

이 문서는 `ceil.py` 파일에 구현된 **올림(Ceiling)** 함수에 대한 설명입니다.

## 개요

**올림 함수**는 실수 `x`를 입력받아 `x`보다 크거나 같은 정수 중 가장 작은 정수를 반환하는 함수입니다. 수학적으로 $\lceil x \rceil$로 표기합니다.

## 함수 설명

### `ceil(x) -> int`

주어진 숫자 `x`의 올림 값을 계산하여 정수 형태로 반환합니다.

#### 매개변수 (Parameters)

- `x`: 올림을 계산할 숫자 (정수 또는 실수).

#### 알고리즘 (Algorithm)

파이썬의 내장 함수 `int()`는 소수점 이하를 0 방향으로 버리는(truncate) 특성이 있습니다. 이를 이용하여 다음과 같이 구현되었습니다.

1. `int(x)`를 통해 `x`의 정수 부분을 구합니다.
2. `x - int(x)` 값을 확인합니다.
   - **0 이하인 경우**: `x`가 정수이거나(차이가 0), 음수이면서 소수 부분이 있는 경우(차이가 음수)입니다. 이때는 `int(x)`가 곧 올림 값이므로 그대로 반환합니다.
     - 예: `ceil(2.0)` -> `2`, `ceil(-1.5)` -> `-1`
   - **0 초과인 경우**: `x`가 양수이면서 소수 부분이 있는 경우입니다. 이때는 `int(x)` 값보다 1 큰 정수가 올림 값이므로 `int(x) + 1`을 반환합니다.
     - 예: `ceil(1.1)` -> `1 + 1` -> `2`
