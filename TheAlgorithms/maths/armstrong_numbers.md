# Armstrong Numbers (암스트롱 수) 알고리즘

이 문서는 `armstrong_numbers.py` 파일에 구현된 **암스트롱 수(Armstrong Number)** 판별 알고리즘에 대한 설명입니다.

## 개요

**암스트롱 수**(또는 **나르시시즘 수(Narcissistic number)**, **플루퍼펙트 수(Pluperfect number)**)란 $n$자리의 자연수 중에서, 각 자리의 숫자를 $n$제곱한 합이 자기 자신과 같은 수를 말합니다.

예를 들어, 370은 3자리 수이며 $3^3 + 7^3 + 0^3 = 27 + 343 + 0 = 370$이므로 암스트롱 수입니다.

## 함수 설명

이 파일은 동일한 기능을 수행하는 세 가지 다른 구현 방식을 제공합니다.

### `armstrong_number(n: int) -> bool`

가장 기본적인 반복문과 산술 연산을 사용하여 구현된 함수입니다.

#### 매개변수 (Parameters)

- `n` (`int`): 판별할 양의 정수입니다.

#### 알고리즘 (Algorithm)

1. 입력값이 정수인지, 1 이상인지 확인합니다. 그렇지 않으면 `False`를 반환합니다.
2. 숫자의 자릿수를 계산합니다.
3. 각 자리의 숫자를 추출하여 자릿수만큼 거듭제곱한 뒤 합산합니다.
4. 합계가 원래 숫자와 같은지 비교하여 결과를 반환합니다.

### `pluperfect_number(n: int) -> bool`

각 숫자의 등장 횟수(히스토그램)를 이용하여 계산을 최적화한 구현입니다.

#### 알고리즘 (Algorithm)

1. 0부터 9까지의 숫자가 몇 번 등장하는지 세어 `digit_histogram` 리스트에 저장합니다.
2. 동시에 전체 자릿수(`digit_total`)를 계산합니다.
3. 히스토그램을 순회하며 `(등장 횟수 * 숫자^자릿수)`를 합산합니다.
4. 합계가 원래 숫자와 같은지 비교합니다.

### `narcissistic_number(n: int) -> bool`

파이썬의 문자열 변환과 리스트 내포(List Comprehension) 기능을 사용하여 간결하게 구현한 함수입니다.

#### 알고리즘 (Algorithm)

1. 숫자를 문자열로 변환하여 길이(자릿수, `expo`)를 구합니다.
2. 문자열의 각 문자를 다시 정수로 변환하고 `expo`만큼 거듭제곱하여 합산합니다.
3. 합계가 원래 숫자와 같은지 비교합니다.
