# Collatz Sequence (콜라츠 수열) 알고리즘

이 문서는 `collatz_sequence.py` 파일에 구현된 **콜라츠 수열(Collatz Sequence)** 생성 알고리즘에 대한 설명입니다.

## 개요

**콜라츠 추측(Collatz conjecture)**은 어떤 자연수에서 시작하더라도 아래의 규칙을 반복하면 결국 1에 도달한다는 수학적 추측입니다.

1. 숫자가 짝수라면 2로 나눕니다.
2. 숫자가 홀수라면 3을 곱하고 1을 더합니다.
3. 결과가 1이 될 때까지 이 과정을 반복합니다.

## 함수 설명

### `collatz_sequence(n: int) -> list[int]`

주어진 양의 정수 `n`에 대해 1에 도달할 때까지의 콜라츠 수열을 리스트로 반환합니다.

#### 매개변수 (Parameters)

- `n` (`int`): 수열을 시작할 양의 정수(자연수)입니다.

#### 예외 처리 (Error Handling)

- **Exception**: 입력값 `n`이 정수가 아니거나 1 미만인 경우 "Sequence only defined for natural numbers" 메시지와 함께 예외를 발생시킵니다.

#### 알고리즘 (Algorithm)

1. 입력값 `n`이 유효한지 검사합니다.
2. 초기값 `n`을 리스트 `sequence`에 저장합니다.
3. `n`이 1이 될 때까지 다음을 반복합니다:
   - `n`이 홀수이면 (`n & 1`이 참이면) `3 * n + 1`을 계산합니다.
   - `n`이 짝수이면 `n // 2`를 계산합니다.
   - 갱신된 `n`을 `sequence`에 추가합니다.
4. 완성된 수열 `sequence`를 반환합니다.

## 테스트 및 실행

파일을 직접 실행하면(`if __name__ == "__main__":`), 숫자 43에 대한 콜라츠 수열을 계산하고 결과를 출력합니다.

```python
if __name__ == "__main__":
    main()
```

**실행 결과 예시:**

```
[43, 130, 65, 196, 98, 49, 148, 74, 37, 112, 56, 28, 14, 7, 22, 11, 34, 17, 52, 26, 13, 40, 20, 10, 5, 16, 8, 4, 2, 1]
collatz sequence from 43 took 30 steps.
```
