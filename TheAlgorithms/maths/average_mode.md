# Average Mode (최빈값) 알고리즘

이 문서는 `average_mode.py` 파일에 구현된 **최빈값(Mode)** 계산 알고리즘에 대한 설명입니다.

## 개요

**최빈값**은 데이터 집합에서 가장 자주 등장하는 값을 의미합니다. 데이터 분포의 대표값 중 하나이며, 숫자뿐만 아니라 문자열 등 다양한 데이터 타입에 대해서도 구할 수 있습니다. 최빈값은 하나일 수도 있고, 여러 개(Multimodal)일 수도 있습니다.

## 함수 설명

### `mode(input_list: list) -> list[Any]`

주어진 리스트에서 최빈값을 찾아 리스트 형태로 반환합니다.

#### 매개변수 (Parameters)

- `input_list` (`list`): 최빈값을 구할 데이터들이 담긴 리스트입니다. 숫자, 문자열 등 다양한 타입을 포함할 수 있습니다.

#### 알고리즘 (Algorithm)

1. 입력 리스트가 비어있는지 확인합니다. 비어있으면 빈 리스트를 반환합니다.
2. 리스트의 각 요소가 리스트 내에서 몇 번 등장하는지 세어 `result` 리스트에 저장합니다. (`input_list.count(value)` 사용)
   - _참고_: 현재 구현은 리스트의 모든 요소에 대해 `count()`를 호출하므로 시간 복잡도가 $O(N^2)$입니다.
3. 등장 횟수 중 최대값(`y`)을 찾습니다.
4. 최대 등장 횟수와 일치하는 요소들을 찾아 집합(Set)으로 중복을 제거하고, 정렬하여 반환합니다.
