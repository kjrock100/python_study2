# 쿨롱의 법칙 (Coulomb's Law)

이 문서는 `coulombs_law.py` 파일에 구현된 **쿨롱의 법칙** 계산 알고리즘에 대해 설명합니다.

## 개요

쿨롱의 법칙은 두 점전하 사이에 작용하는 정전기력의 크기를 설명하는 물리 법칙입니다. 힘의 크기는 두 전하량의 곱에 비례하고, 두 전하 사이의 거리의 제곱에 반비례합니다.

$$F = k_e \frac{|q_1 q_2|}{r^2}$$

- $F$: 정전기력 (Force)
- $k_e$: 쿨롱 상수 ($8.988 \times 10^9 \, N \cdot m^2 \cdot C^{-2}$)
- $q_1, q_2$: 전하량 (Charge)
- $r$: 거리 (Distance)

## 주요 함수: `couloumbs_law`

### `couloumbs_law(force, charge1, charge2, distance)`

- **목적**: 힘, 두 전하량, 거리 중 알 수 없는 하나의 값을 계산합니다.
- **매개변수**:
  - `force`: 정전기력 ($N$).
  - `charge1`: 첫 번째 전하량 ($C$).
  - `charge2`: 두 번째 전하량 ($C$).
  - `distance`: 두 전하 사이의 거리 ($m$).
  - *주의*: 계산하고자 하는 값은 `0`으로 전달해야 합니다. 입력된 4개의 값 중 정확히 하나만 `0`이어야 합니다.
- **반환값**: `{변수명: 계산된 값}` 형태의 딕셔너리.

### 알고리즘 동작 원리

1. **유효성 검사**:
   - 입력된 네 값 중 `0`인 값이 정확히 하나인지 확인합니다. 그렇지 않으면 `ValueError`를 발생시킵니다.
   - 거리는 음수가 될 수 없으므로, `distance < 0`인 경우 `ValueError`를 발생시킵니다.

2. **계산**:
   - `force`가 0인 경우: $F = k_e \frac{|q_1 q_2|}{r^2}$
   - `charge1`이 0인 경우: $|q_1| = \frac{|F| r^2}{k_e |q_2|}$
   - `charge2`가 0인 경우: $|q_2| = \frac{|F| r^2}{k_e |q_1|}$
   - `distance`가 0인 경우: $r = \sqrt{k_e \frac{|q_1 q_2|}{|F|}}$

## 사용법

`if __name__ == "__main__":` 블록은 `doctest`를 실행하여 함수가 올바르게 동작하는지 검증합니다.

예시:
```python
# 힘 계산 (전하량 3C, 5C, 거리 2000m)
couloumbs_law(force=0, charge1=3, charge2=5, distance=2000)
# 결과: {'force': 33705.0}
```
