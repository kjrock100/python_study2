# 부분집합 조합 생성 (Subset Combination Generation)

이 문서는 `subset_generation.py` 파일에 구현된 **부분집합 조합 생성** 알고리즘에 대해 설명합니다. 이 코드는 주어진 $n$개의 원소를 가진 집합에서 $r$개의 원소를 선택하는 모든 조합(Combination)을 생성하고 출력합니다.

## 개요

재귀(Recursion)를 사용하여 각 원소를 선택하거나 선택하지 않는 방식을 통해 크기가 $r$인 모든 부분집합을 찾아냅니다.

## 주요 함수 설명

### `combination_util(arr, n, r, index, data, i)`
- **목적**: 재귀적으로 조합을 생성하고 출력하는 헬퍼 함수입니다.
- **매개변수**:
  - `arr`: 입력 배열 (전체 집합).
  - `n`: 입력 배열의 크기.
  - `r`: 생성할 조합의 크기.
  - `index`: 임시 배열 `data`의 현재 인덱스.
  - `data`: 현재 조합을 저장하고 있는 임시 배열.
  - `i`: 입력 배열 `arr`의 현재 인덱스.
- **알고리즘 동작 원리**:
  1. **출력 조건**: `index`가 `r`에 도달하면, `data` 배열이 꽉 찼다는 의미이므로 현재 조합을 출력하고 반환합니다.
  2. **종료 조건**: `i`가 `n` 이상이면, 더 이상 고려할 원소가 없으므로 반환합니다.
  3. **재귀 호출**:
     - **현재 원소 포함**: `arr[i]`를 `data[index]`에 넣고, `index`와 `i`를 모두 1씩 증가시켜 재귀 호출합니다.
     - **현재 원소 제외**: `index`는 그대로 두고(현재 위치에 다른 값을 덮어쓰기 위해), `i`만 1 증가시켜 재귀 호출합니다.

### `print_combination(arr, n, r)`
- **목적**: 조합 생성 프로세스를 시작하는 래퍼(Wrapper) 함수입니다.
- **동작**:
  - 크기가 `r`인 임시 배열 `data`를 생성합니다.
  - `combination_util` 함수를 호출하여 탐색을 시작합니다. (초기 인덱스 0)

## 사용법

코드 하단에서 사용 예시를 확인할 수 있습니다:

1. 배열 `arr`을 정의합니다: `[10, 20, 30, 40, 50]`
2. `print_combination`을 호출하여 5개 중 3개를 뽑는 모든 조합을 출력합니다.

```python
arr = [10, 20, 30, 40, 50]
print_combination(arr, len(arr), 3)
```

**출력 예시**:
```
10 20 30 
10 20 40 
...
```
