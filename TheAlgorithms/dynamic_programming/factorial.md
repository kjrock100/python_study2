# 팩토리얼 (Factorial)

이 문서는 `factorial.py` 파일에 구현된 **팩토리얼(Factorial)** 계산 알고리즘에 대해 설명합니다. 이 코드는 재귀(Recursion)와 메모이제이션(Memoization)을 사용하여 팩토리얼을 효율적으로 계산합니다.

## 개요

팩토리얼($n!$)은 1부터 $n$까지의 모든 양의 정수의 곱을 의미합니다. 이 구현에서는 Python의 `functools.lru_cache` 데코레이터를 사용하여 이미 계산된 값을 캐싱함으로써 중복 계산을 방지합니다.

## 주요 함수: `factorial(num)`

### `factorial(num: int) -> int`
- **목적**: 주어진 정수 `num`의 팩토리얼을 계산하여 반환합니다.
- **매개변수**:
  - `num`: 팩토리얼을 구할 정수.
- **반환값**: `num!`의 값.
- **예외 처리**: `num`이 음수일 경우 `ValueError`를 발생시킵니다.

### 알고리즘 동작 원리

1. **메모이제이션 (Memoization)**:
   - `@lru_cache` 데코레이터를 사용하여 함수의 반환값을 캐시(저장)합니다.
   - 동일한 입력값으로 함수가 다시 호출되면, 계산을 수행하지 않고 캐시된 값을 반환하여 속도를 높입니다.

2. **유효성 검사**:
   - 입력값 `num`이 0보다 작은지 확인합니다. 음수의 팩토리얼은 정의되지 않으므로 에러를 발생시킵니다.

3. **재귀적 계산**:
   - **기저 사례 (Base Case)**: `num`이 0 또는 1이면 1을 반환합니다. ($0! = 1$, $1! = 1$)
   - **재귀 단계 (Recursive Step)**: `num * factorial(num - 1)`을 호출하여 값을 구합니다.

## 사용법

`if __name__ == "__main__":` 블록은 `doctest`를 실행하여 함수가 올바르게 동작하는지 검증합니다.

예시:
```python
factorial(7)
# 결과: 5040

[factorial(i) for i in range(10)]
# 결과: [1, 1, 2, 6, 24, 120, 720, 5040, 40320, 362880]
```
