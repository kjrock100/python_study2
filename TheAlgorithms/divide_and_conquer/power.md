# 거듭제곱 계산 (Power Calculation)

이 문서는 `power.py` 파일에 구현된 **거듭제곱(Power)** 계산 알고리즘에 대해 설명합니다. 이 코드는 **분할 정복(Divide and Conquer)** 방식을 사용하여 $a^b$를 계산합니다.

## 개요

거듭제곱을 단순히 $b$번 곱하는 방식($O(b)$) 대신, 지수를 절반으로 나누어 재귀적으로 계산하는 방식을 사용합니다.

## 주요 함수 설명

### `actual_power(a: int, b: int)`
- **목적**: 정수 $a$와 $b$에 대해 $a^b$를 재귀적으로 계산합니다.
- **알고리즘**:
  1. **기저 사례 (Base Case)**: $b = 0$이면 1을 반환합니다.
  2. **재귀 단계**:
     - $b$가 짝수인 경우: $a^{b/2} \times a^{b/2}$
     - $b$가 홀수인 경우: $a \times a^{b/2} \times a^{b/2}$
- **주의사항**: 현재 구현에서는 `actual_power(a, int(b / 2))`를 두 번 호출하고 있습니다. 이는 중복 계산을 초래하여 시간 복잡도가 $O(b)$가 됩니다. 이를 한 번만 호출하여 변수에 저장해 사용하면 $O(\log b)$로 최적화할 수 있습니다.

### `power(a: int, b: int) -> float`
- **목적**: 사용자 인터페이스 역할을 하는 래퍼(Wrapper) 함수입니다. 음수 지수를 처리합니다.
- **동작**:
  - $b < 0$인 경우: $1 / \text{actual\_power}(a, b)$를 반환합니다.
    - *참고*: 파이썬의 `int()` 형변환은 0 방향으로 소수점을 버리므로, 음수 입력에 대해서도 재귀 호출이 0으로 수렴하여 절댓값에 대한 거듭제곱이 계산됩니다.
  - 그 외의 경우: `actual_power(a, b)`를 그대로 반환합니다.

## 사용법

`if __name__ == "__main__":` 블록에서 사용 예시를 확인할 수 있습니다.

```python
print(power(-2, -3))
# 결과: -0.125 ( (-2)^-3 = 1/(-8) = -0.125 )
```

## 개선 제안

현재 코드는 분할 정복의 이점을 살리지 못하고 있습니다. 다음과 같이 수정하면 효율성이 크게 향상됩니다:

```python
res = actual_power(a, int(b / 2))
return res * res if b % 2 == 0 else a * res * res
```
