# 슈트라센 행렬 곱셈 (Strassen Matrix Multiplication)

이 문서는 `strassen_matrix_multiplication.py` 파일에 구현된 **슈트라센 알고리즘(Strassen Algorithm)**에 대해 설명합니다. 이 알고리즘은 분할 정복(Divide and Conquer) 기법을 사용하여 두 행렬의 곱을 계산하는 효율적인 방법입니다.

## 개요

일반적인 행렬 곱셈 알고리즘은 $O(n^3)$의 시간 복잡도를 가지지만, 슈트라센 알고리즘은 곱셈 연산 횟수를 줄여 약 $O(n^{2.8074})$의 시간 복잡도를 가집니다. 큰 행렬을 계산할 때 더 빠른 성능을 보입니다.

## 주요 함수 설명

### `split_matrix(a)`
- **목적**: 입력 행렬을 4개의 사분면(Top-Left, Top-Right, Bottom-Left, Bottom-Right)으로 분할합니다.
- **전제**: 행렬의 크기가 짝수여야 합니다.

### `actual_strassen(matrix_a, matrix_b)`
- **목적**: 슈트라센 알고리즘의 핵심 재귀 함수입니다.
- **동작**:
  1. **기저 사례 (Base Case)**: 행렬 크기가 2x2이면 `default_matrix_multiplication`을 사용하여 계산합니다.
  2. **분할**: 입력 행렬을 4개로 나눕니다.
  3. **7번의 곱셈**: 분할된 행렬들의 덧셈과 뺄셈을 조합하여 7개의 중간값(`t1` ~ `t7`)을 재귀적으로 계산합니다.
  4. **결합**: 중간값들을 더하고 빼서 결과 행렬의 4개 사분면을 구성합니다.

### `strassen(matrix1, matrix2)`
- **목적**: 임의의 크기를 가진 행렬에 대해 슈트라센 알고리즘을 적용하기 위한 래퍼(Wrapper) 함수입니다.
- **동작**:
  1. **패딩 (Padding)**: 입력 행렬의 차원을 확인하고, 계산을 용이하게 하기 위해 크기가 2의 거듭제곱이 되도록 0으로 채웁니다.
  2. **계산**: `actual_strassen`을 호출하여 곱셈을 수행합니다.
  3. **복원**: 패딩으로 추가된 0 행과 열을 제거하여 최종 결과 행렬을 반환합니다.

## 사용법

`if __name__ == "__main__":` 블록에서 사용 예시를 확인할 수 있습니다:

1. 두 개의 행렬 `matrix1`과 `matrix2`를 정의합니다.
2. `strassen(matrix1, matrix2)`를 호출하여 곱셈 결과를 출력합니다.

```python
matrix1 = [
    [2, 3, 4, 5],
    [6, 4, 3, 1],
    # ...
]
matrix2 = [[0, 2, 1, 1], [16, 2, 3, 3], ...]
print(strassen(matrix1, matrix2))
```

## 참고 사항

- 이 구현은 행렬의 크기를 2의 거듭제곱으로 맞추기 위해 0을 추가하는 패딩 방식을 사용합니다. 이는 슈트라센 알고리즘의 재귀적 분할을 단순화하기 위함입니다.
