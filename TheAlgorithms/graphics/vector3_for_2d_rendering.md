# 2D 렌더링을 위한 3D 벡터 (Vector3 for 2D Rendering)

이 문서는 `vector3_for_2d_rendering.py` 파일에 구현된 **3D 점의 2D 투영 및 회전** 알고리즘에 대해 설명합니다.

## 개요

3차원 공간의 점을 2차원 화면에 표시하기 위한 투영(Projection) 기능과, 물체를 회전시키기 위한 회전(Rotation) 기능을 제공합니다. 이를 통해 간단한 3D 렌더링 로직을 구현할 수 있습니다.

## 주요 함수

### `convert_to_2d(x, y, z, scale, distance)`

- **목적**: 3D 좌표 $(x, y, z)$를 2D 화면 좌표로 변환합니다.
- **매개변수**:
  - `x`, `y`, `z`: 3D 공간상의 점 좌표 (float 또는 int).
  - `scale`: 투영 시 적용할 확대/축소 비율.
  - `distance`: 시점(카메라)과 투영면 사이의 거리. 원근감(Perspective) 효과를 줍니다.
- **반환값**: 투영된 2D 좌표 `(projected_x, projected_y)`.
- **동작 원리**:
  - 원근 투영 공식을 사용하여 $z$축 깊이에 따라 $x, y$ 좌표를 조정합니다.
  - $z$ 값이 클수록(멀어질수록) 중심점으로 모이는 효과가 나타납니다.

### `rotate(x, y, z, axis, angle)`

- **목적**: 3D 점을 지정된 축을 기준으로 회전시킵니다.
- **매개변수**:
  - `x`, `y`, `z`: 회전할 3D 점의 좌표.
  - `axis`: 회전축 (`'x'`, `'y'`, `'z'` 중 하나).
  - `angle`: 회전할 각도 (0~360).
- **반환값**: 회전된 새로운 3D 좌표 `(new_x, new_y, new_z)`.
- **동작 원리**:
  - 입력된 `angle`을 내부 로직에 따라 라디안 값으로 변환합니다.
  - 선택된 축(`axis`)에 해당하는 회전 행렬 공식을 적용하여 새로운 좌표를 계산합니다.
    - **Z축 회전**: $x, y$ 좌표 변경.
    - **X축 회전**: $y, z$ 좌표 변경.
    - **Y축 회전**: $x, z$ 좌표 변경.

## 사용법

`if __name__ == "__main__":` 블록에서 `doctest`를 실행하여 함수가 올바르게 동작하는지 검증하고 예시를 출력합니다.

```python
# 3D 좌표 (1, 2, 3)을 2D로 변환
convert_to_2d(1.0, 2.0, 3.0, 10.0, 10.0)
# 결과: (7.69..., 15.38...)

# 3D 좌표 (1, 2, 3)을 Y축 기준으로 90도 회전
rotate(1.0, 2.0, 3.0, 'y', 90.0)
# 결과: (3.13..., 2.0, 0.44...)
```

## 주의 사항
- 입력값은 반드시 숫자형(int, float)이어야 하며, 그렇지 않을 경우 `TypeError`가 발생합니다.
- `rotate` 함수의 각도 변환 로직은 일반적인 `degree to radian` 변환과 다르게 구현되어 있어(`angle / 450 * ...`), 예상치 못한 회전 결과가 나올 수 있으므로 주의가 필요합니다.
