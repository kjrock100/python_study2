# `skew_heap.py` 코드 설명

이 문서는 `skew_heap.py` 스크립트에 구현된 스큐 힙(Skew Heap) 자료 구조에 대해 설명합니다.

## 1. 스큐 힙(Skew Heap)이란?

스큐 힙은 **최소 힙(Min-Heap)** 속성을 만족하는 자가 조정(self-adjusting) 이진 트리 기반의 자료 구조입니다. 이 힙의 가장 큰 특징은 **병합(merge)** 연산을 매우 간단하고 효율적으로 수행한다는 점이며, 모든 연산이 이 `merge`를 기반으로 합니다.

**핵심 아이디어**:
-   두 힙을 병합할 때, 두 루트 중 더 작은 값을 가진 노드를 새로운 루트로 삼습니다.
-   그 후, 새로운 루트의 **왼쪽 자식과 오른쪽 자식을 항상 교환(swap)**합니다.
-   교환된 새로운 왼쪽 자식(원래의 오른쪽 자식)과 다른 힙의 루트를 재귀적으로 병합합니다.

이러한 무조건적인 자식 교환 방식은 트리의 균형을 엄격하게 유지하지는 않지만, 분할 상환 분석(amortized analysis)을 통해 모든 연산이 O(log N)의 시간 복잡도를 가짐이 증명되었습니다.

## 2. 클래스 구조

### `SkewNode` 클래스
-   **역할**: 힙을 구성하는 각 노드를 표현합니다.
-   **속성**:
    -   `_value`: 노드가 저장하는 값.
    -   `left`, `right`: 왼쪽, 오른쪽 자식 노드에 대한 참조.
-   **핵심 메서드**:
    -   `merge(root1, root2)`: 두 개의 노드(를 루트로 하는 힙)를 병합하는 정적 메서드입니다. 이 함수가 스큐 힙의 모든 연산의 핵심입니다.

### `SkewHeap` 클래스
-   **역할**: 스큐 힙 전체를 관리하고 사용자에게 인터페이스를 제공합니다.
-   **속성**:
    -   `_root`: 힙의 루트 노드를 가리킵니다.

## 3. 주요 메서드 설명

### `__init__(self, data: Iterable[T] | None = ())`
-   **역할**: 힙을 초기화합니다. 선택적으로 초기 데이터를 받아 힙을 구성할 수 있습니다.
-   **동작**: `data`의 각 항목에 대해 `insert` 메서드를 호출합니다.

### `insert(self, value: T)`
-   **역할**: 힙에 새로운 값을 삽입합니다.
-   **시간 복잡도**: O(log N) (분할 상환)
-   **동작**: 현재 힙의 루트(`self._root`)와 삽입할 값을 가진 새로운 단일 노드 힙을 `SkewNode.merge`를 통해 병합합니다.

### `pop(self) -> T | None`
-   **역할**: 힙에서 가장 작은 값(루트)을 제거하고 반환합니다.
-   **시간 복잡도**: O(log N) (분할 상환)
-   **동작**:
    1.  현재 루트의 값을 저장합니다.
    2.  루트의 왼쪽 자식과 오른쪽 자식을 `SkewNode.merge`를 통해 병합하여 새로운 루트로 만듭니다.
    3.  저장해 둔 원래 루트의 값을 반환합니다.

### `top(self) -> T`
-   **역할**: 힙에서 가장 작은 값을 삭제하지 않고 반환합니다.
-   **시간 복잡도**: O(1)
-   **동작**: `self._root.value`를 즉시 반환합니다.

### `clear(self)`
-   **역할**: 힙의 모든 요소를 제거합니다.
-   **동작**: `self._root`를 `None`으로 설정합니다.

### `__iter__(self) -> Iterator[T]`
-   **역할**: 힙의 모든 요소를 오름차순으로 순회하는 이터레이터(iterator)를 반환합니다. `list(heap)`과 같이 사용할 수 있습니다.
-   **동작**: 힙이 빌 때까지 `pop`을 반복하여 결과를 생성합니다. 이 과정에서 힙이 비워지지만, 순회가 끝나면 다시 원래 상태로 복원합니다.

### `__bool__(self)`
-   **역할**: 힙이 비어있는지 여부를 확인합니다. `if heap:` 과 같은 구문에서 사용됩니다.

## 4. 사용 예제

`doctest`에 포함된 예제는 스큐 힙의 다양한 활용법을 보여줍니다.

```python
# 리스트로부터 힙 생성
sh = SkewHeap([2, 3, 1, 5, 1, 7])

# 이터레이터를 사용하여 정렬된 리스트로 변환
print(list(sh))
# 출력: [1, 1, 2, 3, 5, 7]

# 빈 힙 생성 및 원소 삽입
sh = SkewHeap()
sh.insert(1)
sh.insert(-1)
sh.insert(0)

# 최솟값 확인
print(sh.top())
# 출력: -1

# 최솟값 추출
print(sh.pop())
# 출력: -1

# 다음 최솟값 확인
print(sh.top())
# 출력: 0

# 힙이 비어있는지 확인
print(bool(sh))
# 출력: True

sh.clear()
print(bool(sh))
# 출력: False
```
