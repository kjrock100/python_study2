# `swap_nodes.py` 코드 설명

이 문서는 `swap_nodes.py` 스크립트에 구현된, 단일 연결 리스트(Singly Linked List)에서 두 노드의 데이터를 교환하는 기능에 대해 설명합니다.

## 1. 개요

이 스크립트는 단일 연결 리스트를 생성하고, 주어진 두 데이터 값을 가진 노드를 찾아 그 노드들의 **데이터**를 서로 바꾸는 기능을 제공합니다.

> **중요**: 이 구현은 노드 자체의 연결(포인터)을 바꾸는 것이 아니라, 노드가 담고 있는 **데이터 값만 교환(swap)**합니다.

## 2. 클래스 및 메서드 설명

### `Node` 클래스

-   **역할**: 연결 리스트를 구성하는 각 요소를 표현합니다.
-   **속성**:
    -   `data`: 노드가 저장하는 실제 데이터.
    -   `next`: 다음 노드를 가리키는 참조.

### `LinkedList` 클래스

#### `__init__(self)`
-   **역할**: 빈 연결 리스트를 생성하며, `head`를 `None`으로 초기화합니다.

#### `print_list(self)`
-   **역할**: 리스트의 모든 노드 데이터를 순회하며 화면에 출력합니다.

#### `push(self, new_data: Any)`
-   **역할**: 리스트의 맨 앞에 새로운 데이터를 삽입합니다. (스택의 `push`와 유사)
-   **동작**:
    1.  `new_data`로 새로운 `Node`를 생성합니다.
    2.  새 노드의 `next`가 현재 `head`를 가리키게 합니다.
    3.  `head`를 새 노드로 갱신합니다.

#### `swap_nodes(self, node_data_1, node_data_2)`
-   **역할**: `node_data_1`과 `node_data_2` 값을 가진 두 노드를 찾아, 그 노드들의 `data` 속성을 서로 교환합니다.
-   **시간 복잡도**: O(N), 두 노드를 찾기 위해 리스트를 최대 두 번 순회합니다.
-   **동작**:
    1.  **자기 자신과 교환 방지**: `node_data_1`과 `node_data_2`가 같으면 아무것도 하지 않고 즉시 종료합니다.
    2.  **첫 번째 노드 찾기**: `head`부터 시작하여 `node_1.data`가 `node_data_1`과 같아질 때까지 리스트를 순회합니다.
    3.  **두 번째 노드 찾기**: `head`부터 다시 시작하여 `node_2.data`가 `node_data_2`과 같아질 때까지 리스트를 순회합니다.
    4.  **예외 처리**: 두 노드 중 하나라도 찾지 못하면(`None`이면) 아무것도 하지 않고 종료합니다.
    5.  **데이터 교환**: 두 노드를 모두 찾았으면, `node_1.data`와 `node_2.data`의 값을 서로 바꿉니다.

## 3. 사용 예제

스크립트의 `if __name__ == "__main__":` 블록은 이 클래스의 사용법을 보여줍니다.

```python
if __name__ == "__main__":
    # 연결 리스트 생성
    ll = LinkedList()

    # 1부터 5까지의 숫자를 리스트 앞에 계속 추가
    # 최종 리스트: 1 -> 2 -> 3 -> 4 -> 5
    for i in range(5, 0, -1):
        ll.push(i)

    # 원본 리스트 출력
    ll.print_list()
    # 출력: 1 2 3 4 5

    # 데이터 1과 4를 가진 노드의 데이터를 교환
    ll.swap_nodes(1, 4)
    print("After swapping")

    # 교환 후 리스트 출력
    ll.print_list()
    # 출력: 4 2 3 1 5
```
