# `binary_tree_mirror.py` 코드 설명

이 문서는 `binary_tree_mirror.py` 스크립트에 구현된 이진 트리 미러링 기능에 대해 설명합니다.

## 1. 이진 트리 미러란?

이진 트리의 미러(mirror)는 트리의 각 노드에서 왼쪽 자식과 오른쪽 자식을 서로 바꾸어 생성된 새로운 트리를 의미합니다. 이 스크립트는 주어진 이진 트리를 미러링하는 기능을 제공합니다.

예를 들어, 다음과 같은 트리가 있다면:

```
      1
     / \
    2   3
   / \
  4   5
```

미러링된 트리는 다음과 같습니다:

```
      1
     / \
    3   2
       / \
      5   4
```

## 2. 자료 구조

이 코드에서는 이진 트리를 파이썬의 **딕셔너리(Dictionary)** 자료 구조를 사용하여 표현합니다.

-   **Key**: 부모 노드의 값 (정수)
-   **Value**: `[왼쪽 자식, 오른쪽 자식]` 형태의 리스트

**예시:**
```python
binary_tree = {1: [2, 3], 2: [4, 5]}
```
위 딕셔너리는 '1'번 노드가 왼쪽 자식 '2'와 오른쪽 자식 '3'을 가지고, '2'번 노드가 왼쪽 자식 '4'와 오른쪽 자식 '5'를 가짐을 의미합니다.

## 3. 함수 설명

### `binary_tree_mirror(binary_tree: dict, root: int = 1) -> dict`

이 함수는 사용자가 직접 호출하는 메인 함수입니다.

-   **역할**: 입력받은 이진 트리를 미러링하고, 그 결과를 새로운 딕셔너리로 반환합니다.
-   **매개변수**:
    -   `binary_tree`: 딕셔너리 형태로 표현된 이진 트리.
    -   `root`: 트리의 시작점이 되는 루트 노드.
-   **동작 과정**:
    1.  입력값의 유효성을 검사합니다 (트리가 비어있는지, 루트 노드가 트리에 존재하는지).
    2.  원본 트리를 변경하지 않기 위해 입력받은 딕셔너리의 복사본을 생성합니다.
    3.  실제 미러링 작업을 수행하는 `binary_tree_mirror_dict` 헬퍼 함수를 호출합니다.
    4.  미러링이 완료된 딕셔너리를 반환합니다.

### `binary_tree_mirror_dict(binary_tree_mirror_dictionary: dict, root: int)`

이 함수는 재귀적으로 트리를 순회하며 각 노드의 자식들을 교환하는 내부 헬퍼 함수입니다.

-   **역할**: 주어진 루트부터 시작하여 모든 하위 노드의 왼쪽/오른쪽 자식을 바꿉니다.
-   **동작 과정 (재귀)**:
    1.  **재귀 종료 조건**: 현재 `root`가 `None`이거나 딕셔너리에 존재하지 않으면(리프 노드의 자식 등) 함수를 종료합니다.
    2.  현재 `root` 노드의 왼쪽 자식과 오른쪽 자식을 서로 교환(swap)합니다.
    3.  원래의 왼쪽 자식과 오른쪽 자식을 각각 새로운 루트로 삼아 재귀적으로 자신을 호출하여 하위 트리도 동일하게 미러링합니다.

## 4. 사용 예제

```python
if __name__ == "__main__":
    binary_tree = {1: [2, 3], 2: [4, 5], 3: [6, 7], 7: [8, 9]}
    print(f"원본 이진 트리: {binary_tree}")
    # 출력: 원본 이진 트리: {1: [2, 3], 2: [4, 5], 3: [6, 7], 7: [8, 9]}

    binary_tree_mirror_dictionary = binary_tree_mirror(binary_tree, 1)
    print(f"미러링된 이진 트리: {binary_tree_mirror_dictionary}")
    # 출력: 미러링된 이진 트리: {1: [3, 2], 2: [5, 4], 3: [7, 6], 7: [9, 8]}
```
