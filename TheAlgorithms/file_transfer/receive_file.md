# 파일 수신 클라이언트 (Receive File Client)

이 문서는 `receive_file.py` 파일에 구현된 **파일 수신 클라이언트** 스크립트에 대해 설명합니다. 이 스크립트는 소켓(Socket) 프로그래밍을 사용하여 서버에 접속하고 파일을 전송받아 로컬에 저장합니다.

## 개요

TCP/IP 소켓을 사용하여 로컬 호스트의 특정 포트(12312)에서 실행 중인 서버에 연결합니다. 연결 후 간단한 인사말을 보내고, 서버로부터 데이터를 받아 `Received_file`이라는 이름의 파일로 저장합니다.

## 코드 동작 과정

1. **소켓 생성 및 연결 설정**:
   - `socket` 모듈을 임포트합니다.
   - `socket.socket()`을 통해 소켓 객체를 생성합니다.
   - `socket.gethostname()`으로 현재 로컬 머신의 이름을 가져오고, 포트 번호는 `12312`로 설정합니다.
   - `sock.connect((host, port))`를 호출하여 서버에 연결을 시도합니다.

2. **서버와의 통신**:
   - `sock.send(b"Hello server!")`: 연결이 성공하면 서버에게 바이트 문자열로 인사말을 보냅니다.

3. **파일 수신 및 저장**:
   - `open("Received_file", "wb")`: 수신한 데이터를 저장할 파일을 바이너리 쓰기 모드(`wb`)로 엽니다.
   - `while True:` 루프를 통해 데이터를 계속해서 받습니다.
   - `sock.recv(1024)`: 한 번에 최대 1024 바이트씩 데이터를 수신합니다.
   - `if not data: break`: 더 이상 수신할 데이터가 없으면(빈 바이트 객체 반환) 루프를 종료합니다.
   - `out_file.write(data)`: 수신한 데이터를 파일에 씁니다.

4. **종료**:
   - 파일 수신이 완료되면 성공 메시지를 출력하고 `sock.close()`로 소켓 연결을 닫습니다.

## 사용법

이 스크립트는 클라이언트 역할을 하므로, 파일을 전송해 줄 서버 스크립트(예: `send_file.py`)가 먼저 실행되어 있어야 합니다.

```python
# 터미널에서 실행
python receive_file.py
```

## 주의 사항
- `host` 변수가 `socket.gethostname()`으로 설정되어 있어 로컬 환경에서 테스트하도록 되어 있습니다. 다른 컴퓨터의 서버에 접속하려면 해당 서버의 IP 주소를 `host`에 지정해야 합니다.
- 포트 번호 `12312`는 서버 측 코드와 일치해야 합니다.
